令和8年度の入学者選抜に向けた学習資料をまとめたディレクトリです。


## 選考基準の概要

| 選考 | 調査書 | 学力検査 | 特色検査 | 備考 |
|------|--------|---------|---------|------|
| 第1次（定員90%） | 3 | **7** | 2 | 学力検査が最重要 |
| 第2次（定員10%） | 2 | **8** | 2 | 学力検査がさらに重要 |

- **学力検査の重点化:** 数学・理科を2倍
- **調査書の重点化:** 数学・理科・英語を2倍

## 公式資料・過去問リンク

### 特色検査（自己表現検査）

横浜サイエンスフロンティア高等学校の公式サイトで、特色検査の過去問・サンプル問題が公開されています。

- **公式サイト:** https://www.edu.city.yokohama.lg.jp/school/hs/sfh/index.cfm/35,0,95,266,html

### 神奈川県公立高校 学力検査問題

神奈川県教育委員会が公開している学力検査の過去問です。5教科（英語・国語・数学・理科・社会）の問題と解答が掲載されています。

- **神奈川県公式サイト:** https://www.pref.kanagawa.jp/docs/dc4/nyusen/nyusen/gakuryokukensa/r7.html

### 選考基準

- **選考基準PDF:** [20251001-152819.pdf](../20251001-152819.pdf)（本ディレクトリの親フォルダに格納）

## 使い方

1. **00_選考基準まとめ.md** で配点比率と対策の優先順位を確認する
2. **01_学力検査/** の各教科フォルダに、上記リンクからダウンロードした過去問や教材を保存する
3. **02_調査書対策/内申点管理.md** に現在の内申点を記録し、目標を設定する
4. **03_特色検査/** に特色検査の過去問や演習メモを保存する
5. **04_スケジュール/学習計画.md** で週ごとの学習記録をつける
