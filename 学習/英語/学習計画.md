# 英語 学習計画（調査書 重点化2倍）

## 重要度: ★★★★☆（重要）

調査書で2倍の重点化。内申点と学力検査の両方で得点源にしたい教科。

---

## 出題構成と配点（令和7年度実績）

> 出典: [神奈川県公立高等学校入学者選抜 学力検査問題・正答表（令和7年度）](https://www.pref.kanagawa.jp/docs/dc4/nyusen/nyusen/gakuryokukensa/r7.html)

| 大問 | 内容 | 配点 | 形式 |
|------|------|------|------|
| 問1 | リスニング（短い応答・対話・スピーチ） | 21点（7問×3点） | 選択 |
| 問2 | 語彙（適語補充） | 6点（3問×2点） | 選択 |
| 問3 | 文法（適語補充） | 12点（4問×3点） | 選択 |
| 問4 | 語順整序（6語から5語選択） | 16点（4問×4点） | 選択（両方正解で得点） |
| 問5 | 条件付き英作文 | 5点 | 記述 |
| 問6 | 長文読解（スピーチ原稿+グラフ） | 15点（3問×5点） | 選択 |
| 問7 | 実用英語（対話+予約表/チラシ+問合せ） | 10点（2問×5点） | 選択 |
| 問8 | 長文読解（対話文+グラフ） | 15点（3問×5点） | 選択 |

### 分野別配点まとめ

| 分野 | 配点 | 備考 |
|------|------|------|
| リスニング | **21点** | 問1（全体の21%、最大の単独分野） |
| 語彙・文法 | **18点** | 問2(6点) + 問3(12点) |
| 語順整序 | **16点** | 問4（文法+語彙の総合力、部分点なし） |
| 英作文 | **5点** | 問5（条件付き、6語以上1文） |
| 読解 | **40点** | 問6(15点) + 問7(10点) + 問8(15点) |

### 出題の特徴
- 読解が40点で最大の配点。長文は比較的平易だが分量が多い
- 語順整序は「両方正解で得点」のため、部分点なしに注意
- 英作文は指定語句を使い6語以上1文。正答例: "Do you know how it works?"

## 学習ステップ

### Step 1: 基礎固め（語彙・文法で24点確保）
- [ ] 中学3年間の文法事項を総復習する
- [ ] 教科書レベルの単語・熟語を確実に覚える（約1200語）
- [ ] 語順整序の練習（6語から5語を選ぶ形式に慣れる）

### Step 2: 読解力の強化（40点分の対策）
- [ ] 長文を毎日1つ読む習慣をつける
- [ ] グラフ・表・チラシなどの資料と英文を組み合わせた問題の練習
- [ ] 指示語（it, this, thatなど）が何を指すか意識する
- [ ] 文章の要旨をつかむ練習

### Step 3: リスニング・英作文対策
- [ ] リスニング音声を毎日聞く（15分程度）
- [ ] 条件付き英作文の練習（指定語句を使い1文で書く）
- [ ] 疑問文の作り方を確実にする（Do you know...? 等の間接疑問）
- [ ] シャドーイング練習

### Step 4: 過去問演習
- [ ] 神奈川県公立入試の過去問を5年分以上解く
- [ ] 時間配分の目安: 問1(リスニング約10分) → 問2-4(15分) → 問5(3分) → 問6-8(22分)
- [ ] リスニングは本番形式で練習する

---

## 日々の学習の目安

- 平日: 30〜40分
- 休日: 60〜90分
- 単語の暗記とリスニングは毎日少しずつ続ける
