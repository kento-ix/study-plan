# 学習ディレクトリ

神奈川県公立高校入試に向けた、教科別の学習資料をまとめたディレクトリです。

## ディレクトリ構成

```
学習/
├── 国語/
│   ├── 学習計画.md
│   └── サンプル試験/
├── 数学/
│   ├── 学習計画.md          # 出題構成・配点・学習ステップ
│   ├── 数学要点_学習計画.md  # 要点別の単元一覧と理解度チェック
│   ├── 講義/               # 単元ごとの概念解説（学年別）
│   │   ├── 1年/
│   │   ├── 2年/
│   │   └── 3年/
│   ├── 例題/               # 単元別の例題と解説（学年別）
│   │   ├── 1年/
│   │   ├── 2年/
│   │   └── 3年/
│   ├── 練習問題/            # 単元別の練習問題（学年別）
│   │   ├── 1年/
│   │   ├── 2年/
│   │   └── 3年/
│   ├── クイズ/              # 単元別の確認クイズ（学年別）
│   │   ├── 1年/
│   │   ├── 2年/
│   │   └── 3年/
│   └── サンプル試験/
├── 理科/
│   ├── 学習計画.md
│   └── サンプル試験/
├── 社会/
│   ├── 学習計画.md
│   └── サンプル試験/
└── 英語/
    ├── 学習計画.md
    └── サンプル試験/
```

## 使い方

1. 各教科の `学習計画.md` で出題傾向と学習の進め方を確認する
2. `サンプル試験/` の過去問で現在の実力を把握する
3. 数学は以下の順番で進める（詳しくは `数学/学習計画.md` を参照）
   - **Step 1:** `講義/` で概念を理解する
   - **Step 2:** `例題/` で解き方のパターンを覚える
   - **Step 3:** `練習問題/` で自力で解けるか確認する
   - **Step 4:** `クイズ/` で仕上げのテストをする
4. 勉強が終わったら `進捗/日/YYYY-MM-DD.md` に記録する
5. 弱点単元を重点的に演習し、理解度を更新する
