## Q1. 相似

**問題:** △ABCと△DEFが相似で、相似比が3:5のとき、AB = 9cm ならば DE の長さを求めなさい。

<details>
<summary>解答</summary>

AB : DE = 3 : 5
9 : DE = 3 : 5
DE = 15cm

</details>

## Q2. 線分比・相似の定理

**問題:** △ABCで、辺AB上の点Dと辺AC上の点Eについて DE // BC のとき、AD = 4, DB = 6, AE = 3 ならば EC の長さを求めなさい。

<details>
<summary>解答</summary>

AD : DB = AE : EC（平行線と線分の比）
4 : 6 = 3 : EC
EC = 18/4 = 4.5

</details>

## Q3. 中点連結定理

**問題:** △ABCで、辺ABの中点をM、辺ACの中点をNとする。BC = 10cm のとき、MN の長さを求めなさい。

<details>
<summary>解答</summary>

中点連結定理より MN = BC/2 = 10/2 = 5cm
また MN // BC

</details>

## Q4. 相似と面積比・体積比

**問題:** 2つの相似な三角形の相似比が 2:3 のとき、面積比と、もし立体なら体積比をそれぞれ求めなさい。

<details>
<summary>解答</summary>

面積比 = 2² : 3² = 4 : 9
体積比 = 2³ : 3³ = 8 : 27

</details>
