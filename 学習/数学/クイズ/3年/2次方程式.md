## Q1. 2次方程式の解き方（因数分解）

**問題:** 次の方程式を解きなさい。
x² - 7x + 12 = 0

<details>
<summary>解答</summary>

(x - 3)(x - 4) = 0
x = 3, 4

</details>

## Q2. 2次方程式の解き方（解の公式）

**問題:** 次の方程式を解きなさい。
2x² + 3x - 1 = 0

<details>
<summary>解答</summary>

解の公式 x = (-b ± √(b²-4ac)) / 2a より
x = (-3 ± √(9+8)) / 4 = (-3 ± √17) / 4

</details>

## Q3. 2次方程式の解き方（平方完成）

**問題:** 次の方程式を解きなさい。
x² + 6x + 2 = 0

<details>
<summary>解答</summary>

(x + 3)² - 9 + 2 = 0
(x + 3)² = 7
x + 3 = ±√7
x = -3 ± √7

</details>

## Q4. 2次方程式の応用（文章題）

**問題:** ある正方形の1辺の長さを3cm長くすると、面積が54cm²増える。もとの正方形の1辺の長さを求めなさい。

<details>
<summary>解答</summary>

もとの1辺を x cm とすると:
(x + 3)² - x² = 54
x² + 6x + 9 - x² = 54
6x = 45
x = 7.5

もとの1辺は7.5cm

</details>
