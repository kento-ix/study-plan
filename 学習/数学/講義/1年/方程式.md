## 1. 方程式の解き方

### 方程式とは
文字を含む等式で、**特定の値を代入したときに成り立つ**もの。その値を**解**という。

### 等式の性質
方程式を解くための4つのルール:

| 性質 | 内容 |
|------|------|
| 両辺に同じ数をたす | A = B → A + C = B + C |
| 両辺から同じ数を引く | A = B → A - C = B - C |
| 両辺に同じ数をかける | A = B → A × C = B × C |
| 両辺を同じ数で割る | A = B → A ÷ C = B ÷ C（C≠0） |

### 移項
等式の一方の辺にある項を、**符号を変えて**もう一方の辺に移すこと。

```
3x - 7 = 2x + 5
3x - 2x = 5 + 7    ← 2xを左辺に移項（符号変わる）、-7を右辺に移項（符号変わる）
x = 12
```

### 基本的な解き方の手順
1. かっこがあれば展開する
2. 移項して文字の項を左辺、数の項を右辺に集める
3. 同類項をまとめる
4. 両辺を文字の係数で割る

## 2. いろいろな方程式

### 分数を含む方程式
**最小公倍数を両辺にかけて分母を払う。**

```
(2x - 1)/3 = (x + 2)/2

両辺に6（3と2の最小公倍数）をかける:
2(2x - 1) = 3(x + 2)
4x - 2 = 3x + 6
x = 8
```

### 小数を含む方程式
**10倍、100倍して整数にする。**

```
0.3x + 1.2 = 0.5x

両辺を10倍:
3x + 12 = 5x
12 = 2x
x = 6
```

## 3. 比例式

### 比例式とは
a : b = c : d の形の式。**外項の積 = 内項の積** が成り立つ。

```
a : b = c : d → ad = bc
```

```
例: x : 6 = 5 : 3

3x = 30
x = 10
```

## 4. 方程式の文章題

### 解き方の手順
1. **求めるものを x とおく**
2. **等しい数量関係を見つけて方程式を立てる**
3. **方程式を解く**
4. **答えが問題に合っているか確認する**

```
例: ある数の3倍から5を引くと、その数に7を足した数と等しい。

x とおくと: 3x - 5 = x + 7
           2x = 12
           x = 6
確認: 3×6-5 = 13, 6+7 = 13 ✓
```

## 5. 速さの文章題

### 速さの公式
```
距離 = 速さ × 時間
時間 = 距離 ÷ 速さ
速さ = 距離 ÷ 時間
```

### 解き方のポイント
- **表を作って整理する**のが効果的
- 「合計の距離」「合計の時間」で等式を作る

```
例: 家から駅まで2.4km。分速60mで歩き、途中から分速200mで走って合計20分。

| | 速さ | 時間 | 距離 |
|---|---|---|---|
| 歩き | 60 | x | 60x |
| 走り | 200 | 20-x | 200(20-x) |

方程式: 60x + 200(20-x) = 2400
```

## 6. 割合の文章題

### 割合の基本
```
割合 = 比べる量 ÷ もとにする量
比べる量 = もとにする量 × 割合
```

| 表現 | 割合 |
|------|------|
| ○割 | ○/10 |
| ○% | ○/100 |
| ○割引き | 1 - ○/10 |
| ○%増し | 1 + ○/100 |

```
例: 定価の2割引きで1200円。定価は？

定価を x 円とすると:
x × (1 - 0.2) = 1200
0.8x = 1200
x = 1500円
```
