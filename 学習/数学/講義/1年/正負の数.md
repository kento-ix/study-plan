## 1. 素数と素因数分解

### 素数とは
**1より大きい自然数で、1とその数自身でしか割り切れない数**を素数という。

- 素数の例: 2, 3, 5, 7, 11, 13, 17, 19, 23, ...
- 2は唯一の偶数の素数
- 1は素数ではない

### 素因数分解
自然数を素数の積で表すことを**素因数分解**という。

**やり方:** 割り切れる素数で順に割っていく。

```
例: 60を素因数分解する

60 ÷ 2 = 30
30 ÷ 2 = 15
15 ÷ 3 = 5
5 は素数

よって 60 = 2² × 3 × 5
```

### 素因数分解の活用
- **最大公約数:** 共通する素因数をすべてかけ合わせる
- **最小公倍数:** すべての素因数を最大の指数でかけ合わせる

```
例: 12と18の最大公約数・最小公倍数
12 = 2² × 3
18 = 2 × 3²

最大公約数 = 2 × 3 = 6
最小公倍数 = 2² × 3² = 36
```

## 2. 正の数・負の数

### 基本概念
- **正の数:** 0より大きい数（+をつけて表す）→ +3, +0.5
- **負の数:** 0より小さい数（-をつけて表す）→ -2, -1.5
- **0** は正でも負でもない

### 数直線
```
←──┬──┬──┬──┬──┬──┬──┬──→
  -3  -2  -1   0  +1  +2  +3
```
- 右に行くほど大きい
- 左に行くほど小さい

### 絶対値
**数直線上で0からの距離**を絶対値という。記号は | | を使う。

- |+5| = 5
- |-5| = 5
- |0| = 0

## 3. 正負の加法・減法

### 加法（たし算）のルール

**同符号のたし算:** 絶対値をたして、共通の符号をつける
```
(+3) + (+5) = +8
(-3) + (-5) = -8
```

**異符号のたし算:** 絶対値の大きい方から小さい方を引き、大きい方の符号をつける
```
(+7) + (-3) = +4   （7の方が大きいので+）
(-7) + (+3) = -4   （7の方が大きいので-）
```

### 減法（ひき算）のルール
**ひく数の符号を変えて、加法に直す**
```
(+5) - (+3) = (+5) + (-3) = +2
(+5) - (-3) = (+5) + (+3) = +8
(-5) - (+3) = (-5) + (-3) = -8
(-5) - (-3) = (-5) + (+3) = -2
```

## 4. 加法・減法の混ざった計算

### 項の考え方
加法だけの式にして、**正の項**と**負の項**に分ける。

```
例: -4 + 7 - 9 + 2 - 6

正の項: +7, +2 → 合計 9
負の項: -4, -9, -6 → 合計 -19

答え: 9 + (-19) = -10
```

### 計算のコツ
- まず加法だけの式に直す
- 正の項どうし、負の項どうしをまとめる
- 最後に合わせる

## 5. 乗法・除法・累乗

### 乗法（かけ算）のルール

| パターン | 符号 | 例 |
|---------|------|-----|
| 正 × 正 | + | (+3)×(+4) = +12 |
| 負 × 負 | + | (-3)×(-4) = +12 |
| 正 × 負 | - | (+3)×(-4) = -12 |
| 負 × 正 | - | (-3)×(+4) = -12 |

**ポイント:** 負の数が**偶数個**なら結果は**正**、**奇数個**なら結果は**負**

### 除法（わり算）
符号の決め方は乗法と同じ。

```
(-12) ÷ (-3) = +4
(-12) ÷ (+3) = -4
```

### 累乗
同じ数をくり返しかけること。

```
(-2)³ = (-2) × (-2) × (-2) = -8
(-2)⁴ = (-2) × (-2) × (-2) × (-2) = +16
```

**注意:** `(-2)²` と `-2²` は違う！
- `(-2)² = (-2)×(-2) = 4`
- `-2² = -(2×2) = -4`

## 6. 四則計算・分配法則

### 計算の順序
1. **かっこの中**を先に計算する
2. **累乗**を計算する
3. **乗法・除法**を計算する
4. **加法・減法**を計算する

```
例: -3 × (4 - 7) + 12 ÷ (-4)
  = -3 × (-3) + (-3)     ← かっこ内と除法
  = 9 + (-3)              ← 乗法
  = 6                     ← 加法
```

### 分配法則
```
a × (b + c) = a × b + a × c
a × (b - c) = a × b - a × c
```

```
例: -5 × (3 + (-7))
方法1: -5 × (-4) = 20
方法2: -5 × 3 + (-5) × (-7) = -15 + 35 = 20
```
