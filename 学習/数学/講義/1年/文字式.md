## 1. 文字式の表し方

### 文字式のきまり

| ルール | 例 |
|--------|-----|
| かけ算の記号 × を省く | a × b → ab |
| 数字は文字の前に書く | x × 3 → 3x |
| 1は省略する | 1 × x → x（1xとは書かない） |
| -1は-だけ書く | -1 × x → -x |
| 割り算は分数で書く | a ÷ b → a/b |
| 同じ文字のかけ算は累乗で書く | a × a → a² |
| アルファベット順に並べる | b × a → ab |

### 具体例

```
a × b × (-3) ÷ c = -3ab/c

x × x × 5 × y = 5x²y

(a + b) ÷ 2 = (a + b)/2
```

## 2. 代入と式の値

### 代入とは
文字に具体的な数を当てはめること。

### 手順
1. 式を書く
2. 文字を数に置き換える（**負の数はかっこで囲む**）
3. 計算する

```
例: 2x² - 3y に x = -2, y = 3 を代入

2 × (-2)² - 3 × 3
= 2 × 4 - 9
= 8 - 9
= -1
```

**よくある間違い:**
- `2x²` に x = -2 → `2×(-2)² = 2×4 = 8` ✓
- `2x²` に x = -2 → `2×(-2²) = 2×(-4) = -8` ✗（累乗の前にかっこ!）

## 3. 文字式の計算（加減）

### 同類項をまとめる
**文字の部分が同じ項**（同類項）をまとめて計算する。

```
3a + 5b - 7a + 2b
= (3a - 7a) + (5b + 2b)
= -4a + 7b
```

### かっこのある式
```
(3a + 5b) - (7a - 2b)
= 3a + 5b - 7a + 2b    ← マイナスの場合、かっこ内の符号がすべて逆になる
= -4a + 7b
```

**重要:** かっこの前が `-` のとき、中の符号をすべて逆にする。

## 4. 文字式の計算（乗除）

### 乗法

```
3a × 4b = 12ab
(-2x) × 5x = -10x²
```

### 除法（分数にする）

```
6ab ÷ 2a = 6ab/2a = 3b
8x²y ÷ 4xy = 2x
```

### 乗除の混ざった計算

```
6xy ÷ 2x × 3y
= (6xy/2x) × 3y
= 3y × 3y
= 9y²
```

**コツ:** 除法を分数に直してから計算すると間違いにくい。

## 5. 関係を表す式

### 等式
等号 `=` を使って数量の関係を表す式。

```
例: 1個a円のりんご5個の代金は、b円の割引券を使うと
5a - b = (支払う金額)
```

### 不等式
不等号 `<`, `>`, `≦`, `≧` を使って大小関係を表す式。

| 記号 | 読み方 | 意味 |
|------|--------|------|
| < | 小なり | 左 < 右（左が小さい） |
| > | 大なり | 左 > 右（左が大きい） |
| ≦ | 以下 | 左 ≦ 右（左が右と同じか小さい） |
| ≧ | 以上 | 左 ≧ 右（左が右と同じか大きい） |

```
例: 1本a円の鉛筆5本と1冊b円のノート3冊で1500円以下
5a + 3b ≦ 1500
```
