## 1. 度数の分布

### 度数分布表
データを**階級**（区間）に分け、各階級に入るデータの数（**度数**）を表にまとめたもの。

| 用語 | 意味 |
|------|------|
| 階級 | データを区切る区間（例: 10以上20未満） |
| 階級の幅 | 区間の大きさ（例: 10） |
| 階級値 | 階級の真ん中の値（例: 15） |
| 度数 | その階級に入るデータの個数 |

### 相対度数
各階級の度数が全体に占める割合。

```
相対度数 = その階級の度数 ÷ 度数の合計
```

すべての相対度数の合計は **1**（= 100%）になる。

### ヒストグラム（柱状グラフ）
度数分布表をグラフにしたもの。横軸が階級、縦軸が度数。棒の間にすき間がないのが特徴。

### 度数折れ線
ヒストグラムの各棒の上辺の中点を結んだ折れ線グラフ。

## 2. 範囲と代表値

### 範囲（レンジ）
```
範囲 = 最大値 - 最小値
```
データの散らばりの大きさを表す。

### 代表値

| 代表値 | 求め方 | 特徴 |
|--------|--------|------|
| **平均値** | 合計 ÷ データの個数 | 極端な値に影響されやすい |
| **中央値（メジアン）** | データを小さい順に並べた真ん中の値 | 極端な値に影響されにくい |
| **最頻値（モード）** | 最も多く出てくる値 | 度数分布表から読み取りやすい |

### 中央値の求め方
- データが**奇数個**: 真ん中の値
- データが**偶数個**: 真ん中2つの平均

```
例（奇数個）: 3, 5, 7, 8, 10 → 中央値 = 7
例（偶数個）: 3, 5, 7, 8 → 中央値 = (5+7)/2 = 6
```

## 3. 近似値

### 近似値と誤差
測定値はつねに近似値であり、**真の値**との差を**誤差**という。

```
誤差 = 近似値 - 真の値
```

### 有効数字
近似値の中で**信頼できる数字**の部分。

```
例: 測定値 3.45
有効数字: 3, 4, 5 の3桁
```

### 有効数字の表し方
(整数部分が1桁の小数) × (10のべき乗) で表す。

```
例: 2400（有効数字2桁）→ 2.4 × 10³
例: 0.0056（有効数字2桁）→ 5.6 × 10⁻³
```

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| 度数分布表 | 階級・度数・階級値の意味 |
| 相対度数 | 度数÷合計、全部足すと1 |
| 平均値 | 合計÷個数、極端な値に弱い |
| 中央値 | 真ん中の値、偶数個のときは2つの平均 |
| 最頻値 | 最も多く出現する値 |
| 有効数字 | a × 10ⁿ の形で表す |
