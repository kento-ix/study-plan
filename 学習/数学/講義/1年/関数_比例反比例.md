# 関数（比例・反比例）

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

---

## 1. 関数とは

ある2つの変数 x, y があって、**xの値を決めるとyの値がただ1つ決まる**とき、「yはxの関数である」という。

---

## 2. 比例

### 定義
yがxに比例するとは、**y = ax**（a ≠ 0）の形で表せること。

- **a** を**比例定数**という
- x = 0 のとき y = 0（必ず原点を通る）

### 比例定数の求め方
```
y = ax に x, y の値を代入して a を求める。

例: x = 3 のとき y = -12
-12 = 3a → a = -4
よって y = -4x
```

### 比例のグラフ
- **原点を通る直線**
- a > 0 → 右上がり
- a < 0 → 右下がり
- |a| が大きいほど傾きが急

```
a > 0（右上がり）    a < 0（右下がり）
    /                  \
   /                    \
──/──                  ──\──
 /                        \
```

---

## 3. 反比例

### 定義
yがxに反比例するとは、**y = a/x**（a ≠ 0）の形で表せること。

- a を比例定数という
- **xy = a** が常に成り立つ
- x = 0 のときは定義されない

### 比例定数の求め方
```
例: x = 4 のとき y = 3
a = xy = 4 × 3 = 12
よって y = 12/x
```

### 反比例のグラフ（双曲線）
- **原点に関して対称な2つの曲線**（双曲線）
- a > 0 → 第1象限と第3象限を通る
- a < 0 → 第2象限と第4象限を通る
- x軸、y軸に限りなく近づくが、交わらない

---

## 4. 座標

### 座標平面
```
        y軸
        ↑
  第2象限 | 第1象限
  (-,+)  |  (+,+)
─────────┼─────────→ x軸
  第3象限 | 第4象限
  (-,-)  |  (+,-)
        │
```

### 座標の読み方
点の位置を **(x座標, y座標)** で表す。

### 対称な点
| 対称 | 変わる座標 | 例: (3, -2) の対称点 |
|------|-----------|---------------------|
| x軸対称 | y座標の符号が逆 | (3, 2) |
| y軸対称 | x座標の符号が逆 | (-3, -2) |
| 原点対称 | 両方の符号が逆 | (-3, 2) |

---

## まとめ

| 概念 | 式 | グラフの特徴 |
|------|-----|------------|
| 比例 | y = ax | 原点を通る直線 |
| 反比例 | y = a/x | 原点対称の双曲線 |

| ポイント | 内容 |
|---------|------|
| 比例定数の求め方 | 1組のx, yの値を式に代入 |
| グラフの向き | aの正負で決まる |
| 座標の対称 | x軸→yが逆、y軸→xが逆、原点→両方逆 |
