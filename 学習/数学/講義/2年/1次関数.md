# 1次関数

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

---

## 1. 1次関数とは

### 定義
**y = ax + b**（a ≠ 0）の形で表せる関数を1次関数という。

- **a:** 傾き（変化の割合）
- **b:** 切片（y軸との交点のy座標）
- b = 0 のとき y = ax → 比例

### 比例との違い
| | 比例 | 1次関数 |
|---|---|---|
| 式 | y = ax | y = ax + b |
| グラフ | 原点を通る直線 | y軸と(0, b)で交わる直線 |

---

## 2. 変化の割合

### 定義
```
変化の割合 = yの増加量 / xの増加量
```

**1次関数では変化の割合は常に一定で、傾き a に等しい。**

```
例: y = -2x + 7
xが3→7に変化 → 変化の割合 = -2（常に傾きと同じ）
```

これが2次関数との大きな違い（2次関数では変化の割合が一定でない）。

---

## 3. 式の求め方

### パターン1: 傾きと1点がわかるとき
y = ax + b に傾き a と点の座標を代入して b を求める。

```
傾き2、点(3, 8)を通る
y = 2x + b に (3,8) 代入: 8 = 6 + b → b = 2
答え: y = 2x + 2
```

### パターン2: 2点がわかるとき
① 傾きを求める → ② 1点を代入して b を求める

```
(1, 3) と (4, 9) を通る
傾き = (9-3)/(4-1) = 2
y = 2x + b に (1,3) 代入: 3 = 2 + b → b = 1
答え: y = 2x + 1
```

### パターン3: 傾きと切片がわかるとき
そのまま y = ax + b に当てはめる。

## 4. 変域

### xの変域 → yの変域

1次関数は単調増加（a > 0）または単調減少（a < 0）なので、**xの端の値に対応するyの値が変域の端**になる。

```
y = -3x + 5 で -1 ≦ x ≦ 2

x = -1: y = 3+5 = 8
x = 2:  y = -6+5 = -1

→ -1 ≦ y ≦ 8
```

**注意:** a < 0 のとき、x が増えると y は減る（変域の向きが逆転する）。

## 5. 直線の式・平行・交点

### 平行な直線
**傾きが等しい**直線は平行。

```
y = 2x + 1 と y = 2x - 3 は平行（傾き = 2）
```

### 2直線の交点
連立方程式を解いて求める。

```
y = 2x + 1  …①
y = -x + 7  …②

①=②: 2x + 1 = -x + 7
3x = 6 → x = 2
y = 5
交点: (2, 5)
```

### x軸との交点（x切片）
y = 0 として x を求める。

```
y = 2x + 1 で y = 0:
0 = 2x + 1 → x = -1/2
x切片: (-1/2, 0)
```

## 6. 1次関数の応用（動点）

### 動点問題の考え方

1. **場合分け**をする（点の位置が辺のどこにあるか）
2. 各場合で**底辺と高さ**をxで表す
3. 面積をxの式で表す

```
例: 長方形ABCD、AB=6, BC=4。
    点PがA→B→Cと秒速1cmで移動。

【0≦x≦6】PはAB上 → AP = x
  △APDの底辺AD=4, 高さ=x
  y = 1/2 × 4 × x = 2x

【6≦x≦10】PはBC上 → BP = x-6
  △APDの底辺AD=4, 高さ=AB=6
  y = 1/2 × 4 × 6 = 12（一定）
```

### 動点問題のコツ
- 図を描いて点の移動を追う
- 場合分けの**境界**は頂点に来る瞬間
- 各場合で1次関数（直線）になることが多い

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| 式の形 | y = ax + b、a=傾き、b=切片 |
| 変化の割合 | 常に傾きaに等しい（1次関数の特徴） |
| 式の求め方 | 傾き+1点、または2点から求める |
| 変域 | a<0のとき変域の向きが逆 |
| 平行 | 傾きが同じ |
| 交点 | 連立方程式を解く |
| 動点 | 場合分けして各区間で式を作る |
