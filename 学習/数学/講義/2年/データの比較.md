> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

## 1. 四分位数

### 四分位数とは
データを小さい順に並べて**4等分**したときの区切りの値。

| 名称 | 意味 |
|------|------|
| **第1四分位数 (Q1)** | 下位25%の位置の値 |
| **第2四分位数 (Q2)** | 中央値（50%の位置） |
| **第3四分位数 (Q3)** | 上位25%の位置の値 |

### 求め方
1. データを小さい順に並べる
2. 中央値(Q2)を求める
3. 中央値より**下のグループ**の中央値 = Q1
4. 中央値より**上のグループ**の中央値 = Q3

```
例: 2, 5, 7, 8, 10, 12, 15, 18, 20（9個）

中央値(Q2) = 10（5番目）

下位グループ: 2, 5, 7, 8
Q1 = (5+7)/2 = 6

上位グループ: 12, 15, 18, 20
Q3 = (15+18)/2 = 16.5
```

### 四分位範囲
```
四分位範囲 = Q3 - Q1
```
データの中央50%の散らばりを表す。**外れ値の影響を受けにくい**のが特徴。

## 2. 箱ひげ図

### 箱ひげ図の構造

```
最小値    Q1    Q2(中央値)  Q3    最大値
  ├────────┤========|========├────────┤
  ←  ひげ  → ←    箱     → ←  ひげ  →
```

| 部分 | 表すもの |
|------|---------|
| 箱の左端 | 第1四分位数 (Q1) |
| 箱の中の線 | 中央値 (Q2) |
| 箱の右端 | 第3四分位数 (Q3) |
| 箱の幅 | 四分位範囲 |
| 左のひげ | 最小値からQ1まで |
| 右のひげ | Q3から最大値まで |

### 箱ひげ図の読み取り

- **箱の幅が広い** → データの中央50%のばらつきが大きい
- **ひげが長い** → 端のデータが離れている
- **中央値（箱の中の線）の位置** → データの偏りがわかる

### 2つのデータの比較

箱ひげ図を並べて比較するとき、注目するポイント:

1. **中央値の位置** → どちらが全体的に高い/低いか
2. **箱の幅** → どちらがばらつきが大きいか
3. **範囲（ひげの端から端）** → 全体の散らばり
4. **箱の偏り** → データの分布の形

```
例: AクラスとBクラスのテスト結果

Aクラス:  ├──┤====|====├──┤
Bクラス:     ├┤==|========├────┤

→ Bクラスの方が中央値が高く、ばらつきも大きい
```

## 3. 入試での出題ポイント

### よく聞かれる問題

1. **箱ひげ図から四分位数を読み取る**
2. **2つの箱ひげ図を比較して正しい記述を選ぶ**
3. **データから箱ひげ図を描く**

### 注意点

- 箱ひげ図から**平均値**はわからない
- 箱ひげ図から**データの個数**はわからない
- 箱ひげ図から**最頻値**はわからない
- わかるのは: 最小値、Q1、中央値、Q3、最大値、範囲、四分位範囲

---

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| 四分位数 | Q1(下位25%), Q2(中央値), Q3(上位25%) |
| 四分位範囲 | Q3 - Q1、中央50%の散らばり |
| 箱ひげ図 | 5つの値で描く（最小、Q1、Q2、Q3、最大） |
| 読み取り | 平均値・データ数・最頻値は読み取れない |
| 比較 | 中央値の位置と箱の幅に注目 |
