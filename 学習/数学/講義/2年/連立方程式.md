## 1. 連立方程式の解き方

### 連立方程式とは
2つの方程式を同時に満たす x, y の値（**解**）を求めること。

### 加減法
2つの式をたしたり引いたりして、1つの文字を消去する。

```
例:  2x + y = 7  …①
     x - y = 2   …②

①+② → 3x = 9 → x = 3
x=3 を②に代入 → 3-y=2 → y=1

答え: (x, y) = (3, 1)
```

**係数をそろえる場合:**
```
 3x + 2y = 12  …①
 2x + 3y = 13  …②

①×3: 9x + 6y = 36
②×2: 4x + 6y = 26
引く:  5x = 10 → x = 2, y = 3
```

### 代入法
一方の式を y = ... や x = ... の形にして、もう一方に代入する。

```
例:  y = 2x - 1  …①
     3x + y = 9  …②

①を②に代入: 3x + (2x-1) = 9
5x = 10 → x = 2
y = 2(2)-1 = 3
```

**使い分け:** どちらかの式が y = ... の形に近いなら代入法、そうでなければ加減法が楽。

## 2. いろいろな連立方程式

### かっこを含む場合
まずかっこを展開して整理する。

```
2(x + y) - 3y = 7 → 2x + 2y - 3y = 7 → 2x - y = 7
```

### A = B = C の形
2つの等式に分ける。

```
2x - 1 = 3y + 2 = x + y + 3

→ 2x - 1 = 3y + 2  …①
   3y + 2 = x + y + 3 …②
```

## 3. 連立方程式（小数・分数）

### 小数を含む場合
10倍、100倍して整数にする。

```
0.3x + 0.2y = 1.4
→ 10倍: 3x + 2y = 14
```

### 分数を含む場合
分母の最小公倍数をかけて分母を払う。

```
x/2 - y/3 = 1
→ 6倍: 3x - 2y = 6
```

## 4. 連立方程式の文章題

### 解き方の手順
1. 求めるものを x, y とおく
2. 2つの等しい関係を見つけて式を立てる
3. 連立方程式を解く
4. 答えが問題に合っているか確認する

```
例: りんご3個とみかん5個で850円、りんご2個とみかん3個で550円

りんご x 円、みかん y 円:
3x + 5y = 850 …①
2x + 3y = 550 …②

①×2 - ②×3: 10y - 9y = 1700-1650 → y = 50
代入: 2x + 150 = 550 → x = 200
```

## 5. 連立文章題（速さ）

### 速さの問題のパターン

**パターン1: 追いつく問題**
```
A が先に出発し、B が後から追いかける
→ 追いついた時の「距離が等しい」で式を立てる
```

**パターン2: 出会う問題**
```
A, B が向かい合って出発
→ 「2人の距離の合計 = 全体の距離」で式を立てる
```

**パターン3: 往復の問題**
```
行きと帰りで速さが違う
→ 「距離が等しい」と「時間の合計」で2つの式を立てる
```

```
例: A-B間を行き時速4km、帰り時速6kmで往復5時間
距離を d km:
d/4 + d/6 = 5
3d + 2d = 60 → d = 12km
```

## 6. 連立文章題（割合）

### 割合の問題のポイント

| 表現 | 式での表し方 |
|------|------------|
| ○%の食塩水 x g | 食塩の量 = 0.○○ × x |
| 男子が○%増えた | 増えた後 = x × (1 + ○/100) |
| 定価の○割引き | 売値 = x × (1 - ○/10) |

```
例: 5%食塩水Aと10%食塩水Bを混ぜて8%食塩水500g

x + y = 500（重さ）
0.05x + 0.10y = 0.08 × 500（食塩の量）
→ 0.05x + 0.10y = 40
→ 5x + 10y = 4000（100倍）

x + y = 500 より x = 500-y:
5(500-y) + 10y = 4000
2500 + 5y = 4000 → y = 300, x = 200
```

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| 加減法 | 係数をそろえてたす/引く |
| 代入法 | 一方を代入する（y=...の形が近いとき便利） |
| 小数・分数 | まず整数にしてから解く |
| 速さの文章題 | 距離=速さ×時間、表で整理 |
| 割合の文章題 | 「全体の量」と「特定の量」で2式 |
