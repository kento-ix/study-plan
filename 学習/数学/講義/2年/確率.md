# 確率

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

---

## 1. 確率の基本

### 確率とは
ある実験や観察で、特定の事柄がどの程度起こりやすいかを数値で表したもの。

```
確率 = その事柄が起こる場合の数 / 起こりうるすべての場合の数
```

### 確率の範囲
```
0 ≦ 確率 ≦ 1
```
- 確率 0: 絶対に起こらない
- 確率 1: 必ず起こる

### 同様に確からしい
すべての場合が**同じ程度に起こると期待できる**こと。確率を求める前提条件。

## 2. 場合の数の数え方

### 樹形図（ツリー図）
すべての場合を枝分かれで書き出す方法。漏れや重複が起きにくい。

```
コインを2回投げる:

1回目  2回目   結果
 表 ─── 表 → (表,表)
    └── 裏 → (表,裏)
 裏 ─── 表 → (裏,表)
    └── 裏 → (裏,裏)

全4通り
```

### 表（2つのさいころなど）
2つのものの組み合わせは表で整理する。

```
大小2つのさいころ:

    小 1  2  3  4  5  6
大┌──────────────────
1 │ 2  3  4  5  6  7
2 │ 3  4  5  6  7  8
3 │ 4  5  6  7  8  9
4 │ 5  6  7  8  9  10
5 │ 6  7  8  9  10 11
6 │ 7  8  9  10 11 12

全36通り（和を記入）
```

## 3. いろいろな確率

### さいころの確率

```
例: 1つのさいころで3の倍数が出る確率
3の倍数: 3, 6 → 2通り
確率 = 2/6 = 1/3
```

### 2つのさいころ

```
例: 和が7になる確率
(1,6)(2,5)(3,4)(4,3)(5,2)(6,1) → 6通り
確率 = 6/36 = 1/6
```

### カードを引く確率
「同時に引く」と「順番に引いて戻さない」は同じ結果になる。

```
例: 1〜5のカードから2枚引いて和が偶数
全体: ₅C₂ = 10通り
偶数+偶数: (2,4) → 1通り
奇数+奇数: (1,3)(1,5)(3,5) → 3通り
確率 = 4/10 = 2/5
```

## 4. 余事象を使う確率

### 余事象とは
ある事柄Aに対して「Aが起こらない」という事柄を**Aの余事象**という。

```
P(A) + P(Aの余事象) = 1
P(A) = 1 - P(Aの余事象)
```

### 「少なくとも〜」は余事象が便利

```
例: コインを3回投げて少なくとも1回表が出る確率

直接数えるのは大変 → 余事象を使う
余事象 = 「全部裏」= (1/2)³ = 1/8

答え: 1 - 1/8 = 7/8
```

### 余事象が有効な場面
- 「少なくとも1つ〜」
- 「〜でない」
- 場合の数が多くて直接数えにくいとき

## 5. 確率の入試での出題パターン

### パターン1: さいころ2つ
和・差・積・大小関係の条件で確率を求める。6×6=36通りの表を書く。

### パターン2: カード・玉を取り出す
組み合わせを樹形図か表で書き出す。

### パターン3: 条件に合う確率
「AかつB」「AまたはB」の区別に注意。

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| 確率の求め方 | 場合の数 ÷ 全体の場合の数 |
| 樹形図 | 漏れなく数え上げる基本手段 |
| 表 | 2つのさいころなどで有効 |
| 余事象 | 「少なくとも〜」で使う。1-P(余事象) |
| 入試ポイント | 表・樹形図を正確に書くことが最重要 |
