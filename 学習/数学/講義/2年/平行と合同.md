## 1. 平行線の錯角と同位角

### 対頂角
2直線が交わるとき、向かい合う角を**対頂角**という。
**対頂角は常に等しい。**

```
  a / \ b
   /   \
──/─────\──
  \   /
 b \ / a
```

### 同位角と錯角

2直線を1本の直線（**横断線**）が横切るとき:

```
    a  b
   ──┼──    ← 直線ℓ
    c  d

    e  f
   ──┼──    ← 直線m
    g  h
```

| 種類 | 位置 | 例 |
|------|------|-----|
| **同位角** | 同じ側の同じ位置 | a と e, b と f |
| **錯角** | 交互の位置（Z型） | c と f, d と e |

### 平行線の性質
2直線 ℓ // m のとき:
- **同位角は等しい**: a = e
- **錯角は等しい**: c = f

逆も成立: 同位角（または錯角）が等しければ2直線は平行。

## 2. 内角の和・外角の和

### 三角形
- 内角の和 = **180°**
- 外角 = **となり合わない2つの内角の和**

```
    A
   / \
  /   \
 B─────C──→ 外角

∠Aの外角 ではなく、
∠Cの外角 = ∠A + ∠B
```

### 多角形の内角の和

```
n角形の内角の和 = 180° × (n - 2)
```

| 図形 | n | 内角の和 |
|------|---|---------|
| 三角形 | 3 | 180° |
| 四角形 | 4 | 360° |
| 五角形 | 5 | 540° |
| 六角形 | 6 | 720° |

### 多角形の外角の和
**何角形でも外角の和は 360°**

正n角形の1つの外角 = 360° ÷ n

## 3. 合同条件

### 三角形の合同条件（3つ）

| 条件 | 略称 |
|------|------|
| **3組の辺**がそれぞれ等しい | SSS |
| **2組の辺とその間の角**がそれぞれ等しい | SAS |
| **1組の辺とその両端の角**がそれぞれ等しい | ASA |

**注意:** 「2辺と1角」でも、角が「間の角」でなければ合同とは限らない。

## 4. 合同の証明

### 証明の書き方（テンプレート）

```
△___と△___において

仮定より:
  ___ = ___ …①
  ___ = ___ …②

また、___なので:
  ___ = ___ …③

①②③より、（合同条件）がそれぞれ等しいので
△___ ≡ △___
```

### 証明のコツ

1. **仮定から直接わかること**を①②として書く
2. **図の性質から導けること**を③として書く
   - 共通な辺: AB = AB
   - 対頂角: ∠AOB = ∠COD
   - 平行線の錯角・同位角
3. **3つの根拠が合同条件のどれに当たるか**を明示する

### 証明でよく使う根拠

| 根拠 | 使える場面 |
|------|-----------|
| 仮定より | 問題文に書いてある条件 |
| 共通な辺 | 2つの三角形が辺を共有する |
| 対頂角 | 2直線が交わる点で向かい合う角 |
| 平行線の錯角 | 2直線が平行 |
| 二等辺三角形の底角 | 2辺が等しい |
