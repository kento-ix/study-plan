## 1. 式の計算（加減・乗除）

### 多項式の加減

**同類項**（文字の部分が同じ項）をまとめて計算する。

```
(3a + 5b) + (2a - 3b) = 5a + 2b
(3a + 5b) - (2a - 3b) = 3a + 5b - 2a + 3b = a + 8b
```

### 数と多項式の乗除

**分配法則**を使う。

```
3(2a - b) = 6a - 3b
(6a - 3b) ÷ 3 = 2a - b
```

### 複雑な計算

```
3(2a - b) - 2(a - 3b)
= 6a - 3b - 2a + 6b
= 4a + 3b
```

**ポイント:** かっこの前がマイナスのとき、中の符号がすべて逆になる。

## 2. 式の値・代入

### 式を簡単にしてから代入

先に同類項をまとめてから代入する方が計算が楽。

```
例: 3x² + 2x - x² + 5x に x = -2 を代入

まず整理: 2x² + 7x
代入: 2(-2)² + 7(-2) = 8 - 14 = -6
```

### 分数式の代入

```
例: (2a + b)/3 - (a - b)/2 に a = 6, b = -3 を代入

方法1: 先に通分して整理
= (2(2a+b) - 3(a-b))/6
= (4a + 2b - 3a + 3b)/6
= (a + 5b)/6

代入: (6 + 5(-3))/6 = (6-15)/6 = -9/6 = -3/2
```

## 3. 式による説明

### 文字を使った証明の手順

1. **文字でおく**（整数 n, m など）
2. **式を立てる**
3. **変形して目的の形にする**
4. **結論を述べる**

### よく使う表現

| 表したい数 | 文字での表し方 |
|-----------|--------------|
| 偶数 | 2n |
| 奇数 | 2n + 1（または 2n - 1） |
| 連続する整数 | n, n+1, n+2 |
| 連続する偶数 | 2n, 2n+2 |
| 2桁の自然数 | 10a + b（a: 十の位, b: 一の位） |
| 3の倍数 | 3n |

### 例: 連続する3つの整数の和が3の倍数であることの証明

```
連続する3つの整数を n, n+1, n+2 とする。
和 = n + (n+1) + (n+2) = 3n + 3 = 3(n+1)
n+1 は整数なので、3(n+1) は3の倍数である。
よって、連続する3つの整数の和は3の倍数である。
```

## 4. 等式の変形

### 目的の文字について解く

方程式を解くのと同じ要領で、指定された文字以外を移項する。

### 手順
1. 目的の文字を含む項を一方の辺に集める
2. それ以外を反対の辺に移項する
3. 目的の文字の係数で両辺を割る

```
例1: S = 1/2 ah を h について解く
2S = ah        ← 両辺に2をかける
h = 2S/a       ← 両辺をaで割る

例2: 2x - 3y = 6 を y について解く
-3y = 6 - 2x   ← 2xを移項
y = (6 - 2x)/(-3)
y = (-6 + 2x)/3
y = (2x - 6)/3
```
