# 三角形・四角形

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

---

## 1. 二等辺三角形

### 定義と性質
2辺が等しい三角形。

| 性質 | 内容 |
|------|------|
| 底角が等しい | AB = AC → ∠B = ∠C |
| 頂角の二等分線が底辺を垂直に二等分 | 頂角の二等分線 = 底辺の垂直二等分線 |

### 二等辺三角形になる条件
**2つの角が等しい** → 二等辺三角形

### 正三角形
3辺がすべて等しい三角形。すべての角が60°。

---

## 2. 直角三角形

### 直角三角形の合同条件
通常の合同条件3つに加えて、直角三角形特有の条件が**2つ**ある。

| 条件 | 内容 |
|------|------|
| **斜辺と1つの鋭角** | 斜辺とその両端の1つの鋭角がそれぞれ等しい |
| **斜辺と他の1辺** | 斜辺と直角を挟む1辺がそれぞれ等しい |

**注意:** 直角三角形では、直角(90°)が共通なので、残り2つの条件で合同が示せる。

---

## 3. 平行四辺形

### 定義
2組の対辺がそれぞれ平行な四角形。

### 平行四辺形の性質（4つ）

| 性質 |
|------|
| 2組の対辺がそれぞれ等しい |
| 2組の対角がそれぞれ等しい |
| 対角線がそれぞれの中点で交わる |
| となり合う角の和は180° |

### 平行四辺形になるための条件（5つ）

| 条件 |
|------|
| ① 2組の対辺がそれぞれ**平行** |
| ② 2組の対辺がそれぞれ**等しい** |
| ③ 2組の対角がそれぞれ等しい |
| ④ 対角線がそれぞれの中点で交わる |
| ⑤ **1組の対辺が等しくかつ平行** |

**⑤が特に重要:** 「等しい」だけでも「平行」だけでもダメ。両方必要。

### 特別な平行四辺形

| 図形 | 追加条件 |
|------|---------|
| **長方形** | 4つの角がすべて直角 |
| **ひし形** | 4つの辺がすべて等しい |
| **正方形** | 長方形 + ひし形（直角 + 等辺） |

| 図形 | 対角線の特徴 |
|------|------------|
| 長方形 | 対角線の長さが等しい |
| ひし形 | 対角線が垂直に交わる |
| 正方形 | 長さが等しく垂直に交わる |

## 4. 平行と面積

### 等積変形
底辺が共通で、頂点が底辺に平行な直線上にある三角形は、**面積が等しい**。

```
     P ─── Q ─── R     ← 平行な直線
    /|    /|    /|
   / |   / |   / |
  A──┴──B──┴──C       ← 底辺AB上

△PABと△QABと△RABは面積が等しい
（底辺ABが共通、高さが同じ）
```

### 平行四辺形と面積

対角線は平行四辺形を**面積が等しい2つの三角形**に分ける。

```
平行四辺形ABCDの対角線ACについて:
△ABC = △ACD（面積が等しい）

対角線の交点Oから4つの三角形に分けると:
△AOB = △BOC = △COD = △DOA
```

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| 二等辺三角形 | 底角が等しい ↔ 2辺が等しい |
| 直角三角形の合同 | 斜辺+鋭角、斜辺+他の1辺 |
| 平行四辺形の性質 | 対辺・対角・対角線の3つの性質 |
| 平行四辺形の条件 | 5つ（特に⑤「1組が等しく平行」） |
| 特別な平行四辺形 | 長方形・ひし形・正方形と対角線の特徴 |
| 等積変形 | 底辺共通+高さ同じ→面積同じ |
