# 関数 y = ax²

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

---

## 1. 2乗に比例する関数とは

### 定義
**y = ax²**（a ≠ 0）の形で表される関数。

### 特徴
- x = 0 のとき y = 0（必ず原点を通る）
- xの値が2倍 → yの値は4倍（2²倍）
- xの値が3倍 → yの値は9倍（3²倍）
- **x が正でも負でも、y の値は同じ**（x と -x で y が等しい）

```
例: y = 2x²
x = -3: y = 2×9 = 18
x = 3:  y = 2×9 = 18  ← 同じ！
```

---

## 2. グラフ

### グラフの形: 放物線（パラボラ）
- **a > 0:** 下に凸（∪型）、頂点が最小
- **a < 0:** 上に凸（∩型）、頂点が最大

```
a > 0          a < 0
  |  ∪          ─┬─
  | / \         / | \
  |/   \       /  |  \
──┼─────    ──┼─────
  |              ∩
```

### グラフの特徴
- **頂点は原点** (0, 0)
- **y軸について対称**
- |a| が大きいほど**開きが小さい**（急な放物線）
- |a| が小さいほど**開きが大きい**（なだらかな放物線）

---

## 3. 変域

### xの変域 → yの変域

**重要:** 1次関数と違い、変域の求め方に注意が必要。

### パターン1: 変域に0を含まない場合
端点の値を比較するだけ。

```
y = x², 1 ≦ x ≦ 3
x=1: y=1, x=3: y=9
→ 1 ≦ y ≦ 9
```

### パターン2: 変域が0をまたぐ場合（最重要!）
**y の最小値は必ず0**（頂点を通るため）。

```
y = x², -2 ≦ x ≦ 3
x=0: y=0（最小）, x=3: y=9（最大）
→ 0 ≦ y ≦ 9

注意: x=-2のときy=4だが、最大値は9（x=3のとき）
```

### パターン3: a < 0 の場合
上に凸なので最大値と最小値が入れ替わる。

```
y = -x², -2 ≦ x ≦ 3
x=0: y=0（最大）, x=3: y=-9（最小）
→ -9 ≦ y ≦ 0
```

## 4. 変化の割合

### 定義
```
変化の割合 = yの増加量 / xの増加量
```

### 1次関数との違い
- **1次関数:** 変化の割合は常に一定（= 傾き）
- **y = ax²:** 変化の割合は**区間によって変わる**

### 便利な公式

y = ax² で x が p から q に変わるとき:

```
変化の割合 = a(p + q)
```

```
例: y = 2x² で x が 1→4
変化の割合 = 2×(1+4) = 10

検算: y(1)=2, y(4)=32
(32-2)/(4-1) = 30/3 = 10 ✓
```

---

## 5. 入試での頻出パターン

### パターン1: 放物線と直線の交点
y = ax² と y = bx + c の連立方程式を解く。

```
ax² = bx + c
ax² - bx - c = 0
```

### パターン2: 2つの関数のグラフ上の三角形の面積
1. 交点を求める
2. 座標から底辺と高さを読み取る
3. 面積を計算する

### パターン3: 変化の割合が与えられる
a(p + q) = 変化の割合 の式を利用。

---

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| グラフ | 放物線、y軸対称、原点が頂点 |
| a > 0 | 下に凸（∪）、頂点が最小 |
| a < 0 | 上に凸（∩）、頂点が最大 |
| 変域 | 0をまたぐとき、最小値(最大値)は頂点 |
| 変化の割合 | a(p+q) で一発計算 |
| 1次関数との違い | 変化の割合が一定でない |
