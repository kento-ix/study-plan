# 2次方程式

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

---

## 1. 2次方程式とは

### 定義
**ax² + bx + c = 0**（a ≠ 0）の形の方程式を2次方程式という。

解は一般に**2つ**（重解のとき1つ、解なしの場合もある）。

---

## 2. 解き方① 因数分解

最もシンプルで速い方法。因数分解できるなら最優先。

### 手順
1. 右辺を0にする
2. 左辺を因数分解する
3. 各因数 = 0 として解を求める

```
x² - 7x + 12 = 0
(x - 3)(x - 4) = 0
x = 3 または x = 4
```

### パターン

```
x² - 9 = 0 → (x+3)(x-3) = 0 → x = ±3
x² + 6x + 9 = 0 → (x+3)² = 0 → x = -3（重解）
2x² - 6x = 0 → 2x(x-3) = 0 → x = 0, 3
```

---

## 3. 解き方② 平方根の利用

**(x + p)² = q** の形にできるとき。

```
x² = 5 → x = ±√5

(x - 3)² = 7
x - 3 = ±√7
x = 3 ± √7
```

---

## 4. 解き方③ 平方完成

因数分解できないとき、**(x + p)² = q** の形に変形する。

### 手順
1. x² + bx の部分に注目
2. (b/2)² を足して引く
3. (x + b/2)² の形を作る

```
x² + 6x + 2 = 0

x² + 6x = -2
x² + 6x + 9 = -2 + 9    ← (6/2)²=9を両辺に足す
(x + 3)² = 7
x + 3 = ±√7
x = -3 ± √7
```

---

## 5. 解き方④ 解の公式

**どんな2次方程式でも使える万能の公式。**

```
ax² + bx + c = 0 のとき:

        -b ± √(b² - 4ac)
  x = ─────────────────────
              2a
```

### 使い方

```
例: 2x² + 3x - 1 = 0
a=2, b=3, c=-1

x = (-3 ± √(9+8)) / 4
x = (-3 ± √17) / 4
```

### 判別式 D = b² - 4ac
- D > 0: 異なる2つの解
- D = 0: 重解（1つの解）
- D < 0: 解なし（中学範囲）

### どの解法を使うか

```
因数分解できる？ → Yes → 因数分解で解く（最速）
       ↓ No
bが偶数？ → 平方完成が楽かも
       ↓
解の公式を使う（確実）
```

## 6. 2次方程式の応用（文章題）

### 手順
1. 求めるものを x とおく
2. 2次方程式を立てる
3. 解く
4. **問題の条件に合うか確認する**（重要!）

### よくある出題パターン

**面積の問題:**
```
正方形の1辺をxとおく → 面積の関係式 → 2次方程式
※ 長さなので x > 0 の解だけが答え
```

**数の問題:**
```
連続する2つの正の整数で、積が○○
x(x+1) = ○○ → 2次方程式
※ 正の整数なので条件に合う解を選ぶ
```

**動点の問題:**
```
x秒後の面積が○cm²
→ 時間の範囲に合う解を選ぶ
```

```
例: 正方形の1辺を3cm長くすると面積が54cm²増える

(x+3)² - x² = 54
x² + 6x + 9 - x² = 54
6x = 45
x = 7.5cm
```

### 解の吟味（必ずやる！）
- 長さ → 正の値のみ
- 人数 → 正の整数のみ
- 時間 → 正の値で問題の範囲内

## まとめ

| 解法 | 使う場面 | 特徴 |
|------|---------|------|
| 因数分解 | 因数分解できるとき | 最速 |
| 平方根 | (x+p)²=q の形 | シンプル |
| 平方完成 | 因数分解できないとき | 途中経過がわかりやすい |
| 解の公式 | いつでも使える | 万能だが計算が重い |

| ポイント | 内容 |
|---------|------|
| 解の吟味 | 問題の条件（正の数、整数など）に合うか確認 |
| 文章題 | x を決める→式を立てる→解く→確認 |
