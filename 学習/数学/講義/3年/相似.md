# 相似

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)

---

## 1. 相似とは

### 定義
一方を拡大（縮小）すると他方にぴったり重なる2つの図形を**相似**という。記号: ∽

### 相似な図形の性質
- **対応する辺の比がすべて等しい**（= 相似比）
- **対応する角がそれぞれ等しい**

---

## 2. 三角形の相似条件（3つ）

| 条件 |
|------|
| ① **3組の辺の比**がすべて等しい |
| ② **2組の辺の比とその間の角**がそれぞれ等しい |
| ③ **2組の角**がそれぞれ等しい |

**③が最もよく使う。** 三角形の内角の和は180°なので、2つの角が等しければ残り1つも等しい。

```
例: ∠A = ∠D, ∠B = ∠E → △ABC ∽ △DEF
```

---

## 3. 平行線と線分の比

### 基本定理
△ABCで DE // BC のとき:

```
AD : DB = AE : EC
AD : AB = AE : AC = DE : BC
```

```
    A
   / \
  D───E     DE // BC
 /     \
B───────C
```

### 平行線と線分の比（一般）
3本以上の平行線が2直線を切るとき、対応する線分の比は等しい。

```
a // b // c のとき
AB : BC = DE : EF

A──D
│   │
B──E
│   │
C──F
```

---

## 4. 中点連結定理

### 定理
三角形の2辺の中点を結ぶ線分は、残りの1辺に**平行**で、その**半分の長さ**。

```
M, NがAB, ACの中点のとき:
MN // BC
MN = BC/2
```

---

## 5. 相似と面積比・体積比

### 面積比
相似比が m : n のとき:

```
面積比 = m² : n²
```

**理由:** 面積は「長さ×長さ」なので、比が2回かかる。

```
例: 相似比 2:3 → 面積比 = 4:9
```

### 体積比
相似比が m : n のとき:

```
体積比 = m³ : n³
```

**理由:** 体積は「長さ×長さ×長さ」なので、比が3回かかる。

```
例: 相似比 2:3 → 体積比 = 8:27
```

### 面積比の応用

**高さが等しい三角形の面積比 = 底辺の比**

```
△ABDと△ACDで高さが等しいとき:
面積比 = BD : CD
```

## 6. 入試での頻出パターン

### パターン1: 相似の証明
2つの三角形が相似であることを示す（主に条件③を使用）。

**証明テンプレート:**
```
△___と△___において
∠___ = ∠___ …①（共通/対頂角/円周角 など）
∠___ = ∠___ …②（平行線の錯角/同位角 など）
①②より、2組の角がそれぞれ等しいので
△___ ∽ △___
```

### パターン2: 線分の長さを求める
相似比を使って未知の辺の長さを求める。

```
△ABC ∽ △DEF、相似比 3:5
AB = 9 → DE = ?

9 : DE = 3 : 5
DE = 15
```

### パターン3: 面積比からの逆算
面積比がわかっているとき、相似比を求める。

```
面積比 = 16:25 → 相似比 = 4:5
```
