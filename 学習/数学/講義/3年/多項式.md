## 1. 式の展開

### 分配法則による展開
```
(a + b)(c + d) = ac + ad + bc + bd
```

各項にすべての項をかけて、同類項をまとめる。

```
例: (2x + 3)(x - 5)
= 2x·x + 2x·(-5) + 3·x + 3·(-5)
= 2x² - 10x + 3x - 15
= 2x² - 7x - 15
```

### 乗法公式（必ず覚える！）

| 公式 | 展開結果 |
|------|---------|
| **(a + b)²** | a² + 2ab + b² |
| **(a - b)²** | a² - 2ab + b² |
| **(a + b)(a - b)** | a² - b² |
| **(x + a)(x + b)** | x² + (a+b)x + ab |

```
例: (x + 4)² = x² + 8x + 16
例: (x + 3)(x - 3) = x² - 9
例: (x + 2)(x + 5) = x² + 7x + 10
```

### 展開の応用

```
例: (x + 4)² - (x - 2)(x + 2)
= (x² + 8x + 16) - (x² - 4)
= x² + 8x + 16 - x² + 4
= 8x + 20
```

## 2. 因数分解のやり方

### 因数分解とは
式の展開の逆操作。多項式を**積の形**に変換すること。

### 基本パターン（4つ）

| パターン | 因数分解 |
|---------|---------|
| **共通因数** | ma + mb = m(a + b) |
| **和と差の積** | a² - b² = (a + b)(a - b) |
| **完全平方式** | a² + 2ab + b² = (a + b)² |
| **たすきがけ** | x² + (a+b)x + ab = (x + a)(x + b) |

### 因数分解の手順

**まず共通因数がないか確認** → あれば先にくくり出す → 公式を適用

```
例1: x² - 5x + 6
→ かけて6、たして-5 → -2と-3
→ (x - 2)(x - 3)

例2: 4x² - 9
→ (2x)² - 3² → (2x + 3)(2x - 3)

例3: x² + 6x + 9
→ x² + 2·3·x + 3² → (x + 3)²

例4: 2x² + 6x
→ 2x(x + 3)  ← 共通因数2xをくくり出す
```

### たすきがけのコツ
x² + (a+b)x + ab の形で:
- **積が定数項(ab)** になる2数を探す
- その2数の**和が一次の係数(a+b)** になるか確認

```
x² - 5x + 6
積が6: (1,6), (2,3), (-1,-6), (-2,-3)
和が-5: → -2と-3 ✓
答え: (x-2)(x-3)
```

## 3. 因数分解のくふう

### 置き換え
共通部分を1つの文字に置き換える。

```
例: (x + y)² - 4(x + y) + 4
A = x + y とおくと:
A² - 4A + 4 = (A - 2)² = (x + y - 2)²
```

### 一部を先にまとめる

```
例: x² + 4xy + 4y² - 9
= (x + 2y)² - 9       ← x² + 4xy + 4y² を完全平方式に
= (x + 2y)² - 3²
= (x + 2y + 3)(x + 2y - 3)
```

### 複雑な因数分解の手順
1. 共通因数をくくり出す
2. 公式が使えるか確認
3. 置き換えや並べ替えを試す
