## 1. 円周角の定理

### 用語
- **中心角:** 弧の両端と**円の中心**を結んでできる角
- **円周角:** 弧の両端と**円周上の点**を結んでできる角

```
        P（円周上の点）
       / \
      /   \  ← 円周角
     /     \
    A───────B
     \     /
      \ O /   ← 中心角
       \ /
```

### 定理

**① 円周角は中心角の半分**
```
円周角 = 中心角 / 2
```

**② 同じ弧に対する円周角は等しい**
弧ABに対して、円周上のどこに点をとっても円周角は同じ値。

```
    P   Q   R  （どこでも同じ角度）
   /|\ /|\ /|\
  / | X | X | \
 A──┴──┴──┴──B

∠APB = ∠AQB = ∠ARB
```

**③ 半円の弧に対する円周角は90°**
直径に対する円周角は必ず直角。

```
        P
       /|\
      / | \  ← 90°
     /  |  \
    A───O───B  ← 直径
```

## 2. 円に内接する四角形

### 性質
円に内接する四角形の**対角の和は180°**。

```
∠A + ∠C = 180°
∠B + ∠D = 180°
```

### 理由
∠Aと∠Cは、それぞれ弧BCD、弧BADに対する円周角。
2つの弧を合わせると円全体(360°)なので、
中心角の合計 = 360° → 円周角の合計 = 180°。

## 3. 円周角の応用

### 円周角の逆
**2点A, Bに対して同じ側にある2点P, Qで ∠APB = ∠AQB ならば、4点A, B, P, Qは1つの円上にある。**

これは証明問題で「4点が同一円周上にある」ことを示すのに使う。

### 接線

**接線は接点を通る半径に垂直。**

```
      |
   ───O───
      |
      | ← 半径
      |
──────P────── ← 接線
```

### 接線と弦の角
接線と弦が作る角は、その弦に対する円周角に等しい（**接弦定理**）。

```
        Q（円周上）
       /
      /
     /  ← ∠AQB = ∠TBA
    A───B
        |
        T（接線方向）
```

## 4. 入試での頻出パターン

### パターン1: 角度を求める
円周角の定理を使って未知の角度を求める。
- 同じ弧に対する円周角は等しい
- 中心角は円周角の2倍

### パターン2: 相似の証明で使う
2つの三角形が相似であることを示すとき、「同じ弧に対する円周角が等しい」ことを根拠にする。

```
∠ACB = ∠ADB（弧ABに対する円周角）
```

### パターン3: 三平方の定理との融合
円周角90°（直径に対する円周角）を利用して直角三角形を作り、三平方の定理を適用する。

## まとめ

| 概念 | 覚えるべきポイント |
|------|------------------|
| 円周角 = 中心角の半分 | 最も基本的な定理 |
| 同じ弧→同じ円周角 | 相似の証明で多用 |
| 直径→円周角90° | 直角三角形が作れる |
| 内接四角形 | 対角の和 = 180° |
| 接線 | 半径に垂直 |
| 接弦定理 | 接線と弦の角 = 円周角 |
