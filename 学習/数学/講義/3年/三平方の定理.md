## 1. 三平方の定理（ピタゴラスの定理）

### 定理
直角三角形で、**直角を挟む2辺を a, b、斜辺を c** とすると:

```
a² + b² = c²
```

```
    |\
  a | \ c（斜辺=最も長い辺）
    |  \
    |___\
      b
```

### 基本的な使い方

**斜辺を求める:**
```
a=3, b=4 → c² = 9+16 = 25 → c = 5
```

**他の辺を求める:**
```
a=?, b=12, c=13 → a² = 169-144 = 25 → a = 5
```

### よく出る整数の組（ピタゴラス数）
| a | b | c |
|---|---|---|
| 3 | 4 | 5 |
| 5 | 12 | 13 |
| 8 | 15 | 17 |
| 7 | 24 | 25 |

**倍数も成り立つ:** (6, 8, 10), (9, 12, 15) など

## 2. 特別な直角三角形

### 必ず覚える2パターン

**① 45°-45°-90°（直角二等辺三角形）**
```
辺の比 = 1 : 1 : √2
```

```
    |\
  1 | \ √2
    |45°\
    |____\
      1
```

**② 30°-60°-90°**
```
辺の比 = 1 : √3 : 2
（30°の対辺 : 60°の対辺 : 斜辺）
```

```
    |\
    | \
 √3 |  \ 2（斜辺）
    |   \
    |60° \
    |_____\
      1
```

### 使い方
辺の長さが1つわかれば、比を使って他の辺も求められる。

```
例: 45-45-90で斜辺が6
1:1:√2 = ?:?:6
各辺 = 6/√2 = 3√2
```

## 3. 平面での利用

### 2点間の距離

```
A(x₁, y₁) と B(x₂, y₂) の距離:
AB = √((x₂-x₁)² + (y₂-y₁)²)
```

```
例: A(1,2) と B(4,6)
AB = √(9+16) = √25 = 5
```

### 正三角形の高さと面積

1辺が a の正三角形:
```
高さ = (√3/2) × a
面積 = (√3/4) × a²
```

```
例: 1辺6cmの正三角形
高さ = 3√3 cm
面積 = 9√3 cm²
```

### 座標上の図形の面積
座標で頂点が与えられた三角形や四角形の面積を求める問題。
→ 底辺と高さを座標から読み取り、三平方の定理で距離を計算。

## 4. 立体での利用

### 直方体の対角線

```
縦a, 横b, 高さc の直方体:
対角線 = √(a² + b² + c²)
```

```
例: 3×4×5の直方体
対角線 = √(9+16+25) = √50 = 5√2
```

### 円錐の母線・高さ・半径

```
    /|\
   / | \
  / h|  \ ℓ（母線）
 /   |   \
/____|____\
     r

ℓ² = r² + h²
```

```
例: 半径3, 高さ4の円錐 → 母線 = √(9+16) = 5
```

### 角錐の高さ
底面の中心から頂点への距離。底面の対角線の半分と三平方の定理で求める。

### 正四面体

1辺 a の正四面体:
```
高さ = √(2/3) × a = a√6 / 3
体積 = (√2/12) × a³
```

## 5. 入試での頻出パターン

### パターン1: 平面図形での利用
- 正三角形の面積
- 二等辺三角形に垂線を引いて高さを求める
- 台形の面積

### パターン2: 座標上の距離
- 2点間の距離
- 放物線と直線の交点間の距離

### パターン3: 空間図形での利用（最難関）
- 直方体・三角柱の対角線
- 円錐・角錐の高さや母線
- 立体の切断面の面積
- **最短距離（展開図にして直線で考える）**

```
例: 円柱の側面上の最短距離
→ 側面を展開して長方形にする
→ 2点を直線で結ぶ
→ 三平方の定理で距離を求める
```
