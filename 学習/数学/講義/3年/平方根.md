## 1. 平方根とは

### 定義
x² = a を満たす x を「aの**平方根**」という。

- a > 0 のとき、平方根は**正と負の2つ** → ±√a
- a = 0 のとき、平方根は **0** のみ
- a < 0 のとき、平方根は**存在しない**（中学範囲）

```
例: 9の平方根 → ±3（3²=9, (-3)²=9）
例: 5の平方根 → ±√5
```

### √ の意味
- **√a** は a の平方根のうち**正の方**（√a ≧ 0）
- 9の平方根は ±3 だが、√9 = 3（正の方だけ）

### 注意
```
√a² = |a|（絶対値）

√3² = √9 = 3
√(-3)² = √9 = 3（-3 ではない!）
```

## 2. 平方根の大小

### √ のついた数の大小比較
a > 0, b > 0 のとき:

```
a < b ならば √a < √b
```

### 比較の方法
√ の中を比較するか、2乗して比較する。

```
例: 3 と √10 の大小
3 = √9 なので、√9 と √10 を比較
9 < 10 → √9 < √10 → 3 < √10
```

### 整数部分の求め方
```
例: √7 の整数部分
2² = 4, 3² = 9 → 4 < 7 < 9 → 2 < √7 < 3
整数部分は 2
```

## 3. ルートの計算

### √ の変形

```
√(a²×b) = a√b（a > 0）
```

```
例: √12 = √(4×3) = 2√3
例: √75 = √(25×3) = 5√3
例: √0.5 = √(1/2) = √2/2
```

### 加法・減法
**√ の中が同じもの**どうしだけ計算できる。

```
2√3 + 5√3 = 7√3
√12 + √27 - √3 = 2√3 + 3√3 - √3 = 4√3
```

**√ の中が違うものは足し引きできない:** √2 + √3 ≠ √5

### 乗法・除法

```
√a × √b = √(ab)
√a ÷ √b = √(a/b)
```

```
例: √6 × √2 = √12 = 2√3
例: √15 ÷ √3 = √5
例: 2√3 × 3√5 = 6√15
```

### 乗法公式の利用

```
(√5 + √2)(√5 - √2) = (√5)² - (√2)² = 5 - 2 = 3
(√3 + 1)² = 3 + 2√3 + 1 = 4 + 2√3
```

## 4. 分母の有理化

### 有理化とは
分母から√を取り除くこと。

### パターン1: 分母が √a

```
b/√a = b×√a / (√a×√a) = b√a / a
```

```
例: 6/√3 = 6√3/3 = 2√3
```

### パターン2: 分母が a + √b または a - √b

```
1/(√3+1) = (√3-1)/((√3+1)(√3-1)) = (√3-1)/(3-1) = (√3-1)/2
```

**ポイント:** (a+b)(a-b) = a²-b² を利用して√を消す。

```
例: 6/(√3+1) = 6(√3-1)/((√3)²-1²) = 6(√3-1)/2 = 3(√3-1) = 3√3-3
```
