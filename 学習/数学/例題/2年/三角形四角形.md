### 二等辺三角形

📖 [二等辺三角形の角](https://math.005net.com/reidai/nitohen.php)

**例題1:** 二等辺三角形ABCで AB = AC, ∠B = 65° のとき、∠A を求めなさい。

<details>
<summary>解答</summary>

∠B = ∠C = 65°（底角は等しい）
∠A = 180° - 65° - 65° = **50°**

</details>

---

📖 [二等辺三角形の性質を利用した証明](https://math.005net.com/reidai/nitohenShomei1.php)

**例題2:** AB = AC の二等辺三角形ABCで、∠Bの二等分線と辺ACの交点をD、∠Cの二等分線と辺ABの交点をEとする。BD = CE を証明しなさい。

<details>
<summary>解答</summary>

△BDCと△CEBにおいて
BC = BC …①（共通）
∠DBC = ∠ECB …②（∠B = ∠C の二等分線なので半分も等しい）
∠BCD = ∠CBE …③（②と同じ理由）
①②③より、1組の辺とその両端の角がそれぞれ等しいので
△BDC ≡ △CEB
よって BD = CE

</details>

### 直角三角形

📖 [直角三角形の合同証明](https://math.005net.com/reidai/chokakuGodo.php)

**例題3:** ∠C = 90° の直角三角形ABCと ∠F = 90° の直角三角形DEFで、AB = DE, BC = EF のとき、△ABC ≡ △DEF を証明しなさい。

<details>
<summary>解答</summary>

△ABCと△DEFにおいて
∠C = ∠F = 90° …①
AB = DE …②（斜辺が等しい）
BC = EF …③（他の1辺が等しい）
①②③より、直角三角形の斜辺と他の1辺がそれぞれ等しいので
△ABC ≡ △DEF

</details>

### 平行四辺形

📖 [平行四辺形の性質](https://math.005net.com/reidai/heikoShihenSeisitu.php)

**例題4:** 平行四辺形ABCDで、∠A = 70° のとき、∠B, ∠C, ∠D をそれぞれ求めなさい。

<details>
<summary>解答</summary>

∠B = 180° - 70° = **110°**（隣り合う角の和は180°）
∠C = **70°**（対角は等しい）
∠D = **110°**（対角は等しい）

</details>

---

📖 [平行四辺形になるための証明](https://math.005net.com/reidai/tmpHeikoshihenNaru.php)

**例題5:** 四角形ABCDで AB = CD, AB // CD のとき、ABCDが平行四辺形であることを証明しなさい。

<details>
<summary>解答</summary>

対角線ACをひく。
△ABCと△CDAにおいて
AB = CD …①（仮定）
AC = CA …②（共通）
∠BAC = ∠DCA …③（AB // CD の錯角）
①②③より △ABC ≡ △CDA
対応する辺より BC = DA
AB = CD（仮定）と合わせて、2組の対辺がそれぞれ等しいので
ABCDは平行四辺形である。

</details>

### 等積変形

📖 [等積変形1](https://math.005net.com/reidai/toseki.php)

**例題6:** △ABCで、辺BC上に点Dをとる。△ABDと面積が等しい三角形を、頂点Aを通る辺BCに平行な直線を利用して作図しなさい。

<details>
<summary>解答</summary>

Aを通りBCに平行な直線上に点Eをとると、
△ABDと△EBDは底辺BDが共通で高さが等しい（平行線間の距離）。
よって △ABD = △EBD（面積が等しい）

</details>

---

📖 [平行線と面積](https://math.005net.com/reidai/heikoMenseki.php)

**例題7:** 平行四辺形ABCDの対角線BDと辺の延長線上の点Eについて、△ABE = △ABD（面積）を示しなさい。

<details>
<summary>解答</summary>

平行四辺形の対角線は面積を二等分するので
△ABD = 平行四辺形ABCD の 1/2

底辺ABが共通で、DとEが辺ABに平行な直線上にあれば
△ABD = △ABE（等積変形）

</details>

📖 [面積の問題(等積変形)](https://math.005net.com/reidai/chohokeiMenseki.php)

**例題8:** 長方形ABCDで AB = 6, BC = 8 とする。辺AB上に点E、辺CD上に点Fをとり、AE = 2, CF = 3 とする。四角形AEFDの面積を求めなさい。

<details>
<summary>解答</summary>

四角形AEFD = △AEF + △AFD
または長方形 - △BEF - △BCF で求める。

△ADF: 底辺AD = 8, 高さ = AB = 6 → ×
台形AEFDとして: 上底AE = 2, 下底DF = 8-3 = 5, 高さ = 8

面積 = 1/2 × (2 + 5) × 8 = **28**

</details>
