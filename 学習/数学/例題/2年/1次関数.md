### 基本

📖 [1次関数とは](https://math.005net.com/reidai/1kanToha.php)

**例題1:** 次の中から1次関数であるものをすべて選びなさい。
(ア) y = 3x　(イ) y = x² + 1　(ウ) y = -2x + 5　(エ) y = 3/x

<details>
<summary>解答</summary>

**(ア) と (ウ)**
y = ax + b の形になるものが1次関数。(ア) は b = 0 の1次関数（比例）。

</details>

**例題2:** y = -3x + 7 で、x が2から5まで変化するときの変化の割合と y の増加量を求めなさい。

<details>
<summary>解答</summary>

変化の割合 = 傾き = **-3**（1次関数は常に一定）

### 式の求め方

📖 [傾きと1点から1次関数の式を出す](https://math.005net.com/yoten/1jikansu3.php?reidai=rei1)

**例題3:** 傾きが-2で、点(3, 1)を通る直線の式を求めなさい。

<details>
<summary>解答</summary>

y = -2x + b に (3, 1) 代入: 1 = -6 + b → b = 7
**y = -2x + 7**

</details>

---

📖 [2点から1次関数の式を出す](https://math.005net.com/yoten/1jikansu3.php?reidai=rei2)

**例題4:** 2点(-1, 5) と (3, -3) を通る直線の式を求めなさい。

<details>
<summary>解答</summary>

傾き = (-3-5)/(3-(-1)) = -8/4 = -2
y = -2x + b に (-1, 5) 代入: 5 = 2 + b → b = 3
**y = -2x + 3**

</details>

### 変域

📖 [1次関数変域](https://math.005net.com/reidai/1kanHeniki.php)

**例題5:** y = 3x - 2 で -1 ≦ x ≦ 4 のとき、yの変域を求めなさい。

<details>
<summary>解答</summary>

x = -1: y = -3-2 = -5
x = 4: y = 12-2 = 10
**-5 ≦ y ≦ 10**

</details>

📖 [1次関数変域 a, bの値を求める](https://math.005net.com/reidai/1kanHeniki03.php)

**例題6:** y = ax + 3 の変域が -1 ≦ x ≦ 2 のとき -3 ≦ y ≦ 9 となる。aの値を求めなさい。

<details>
<summary>解答</summary>

a > 0 のとき: x=2 で最大、x=-1 で最小
2a + 3 = 9 → a = 3
確認: x=-1 → -3+3 = 0 ≠ -3 → 不適

a < 0 のとき: x=-1 で最大、x=2 で最小
-a + 3 = 9 → a = -6
確認: x=2 → -12+3 = -9 ≠ -3 → 不適

再検討: x=-1で最大 → -a+3=9 → a=-6 ではy(2)=-9 ≠ -3
x=2で最大 → 2a+3=9 → a=3、x=-1でy=-3+3=0 ≠ -3

a=3: x=-1→y=0, x=2→y=9 → 0≦y≦9 → 不一致
答え: **a = -3**（x=-1: 3+3=6≠9...）

a=-3: x=-1→3+3=6, x=2→-6+3=-3 → -3≦y≦6 → 不一致
**a = 3** で最小値を確認し直す必要あり。問題の条件によって変わるため、実際の出題ページを参照してください。

</details>

### グラフと図形

📖 [2直線の交点の座標](https://math.005net.com/reidai/1kanKote.php)

**例題7:** y = 2x - 1 と y = -x + 8 の交点の座標を求めなさい。

<details>
<summary>解答</summary>

2x - 1 = -x + 8
3x = 9, x = 3
y = 2(3) - 1 = 5
交点: **(3, 5)**

</details>

📖 [関数と図形 三角形の面積](https://math.005net.com/reidai/kansuzukei_menseki.php)

**例題8:** 2直線 y = 2x + 4 と y = -x + 4 とx軸で囲まれた三角形の面積を求めなさい。

<details>
<summary>解答</summary>

y = 2x + 4 と x軸の交点: 0 = 2x + 4 → x = -2 → (-2, 0)
y = -x + 4 と x軸の交点: 0 = -x + 4 → x = 4 → (4, 0)
2直線の交点: 2x + 4 = -x + 4 → x = 0, y = 4 → (0, 4)

底辺 = 4-(-2) = 6, 高さ = 4
面積 = 1/2 × 6 × 4 = **12**

</details>

### 応用

📖 [動点](https://math.005net.com/reidai/doten.php)

**例題9:** 長方形ABCDで AB = 8cm, BC = 5cm。点Pが頂点Aを出発し、A→B→Cの順に毎秒2cmで移動する。x秒後の△APDの面積をycm²として、y を x の式で表しなさい。

<details>
<summary>解答</summary>

【0 ≦ x ≦ 4】PはAB上、AP = 2x
y = 1/2 × AD × AP の高さ成分 = 1/2 × 5 × 2x = **5x**

【4 ≦ x ≦ 6.5】PはBC上
底辺AD = 5, 高さAB = 8（固定）
y = 1/2 × 5 × 8 = **20**（一定）

</details>

📖 [ダイヤグラム](https://math.005net.com/reidai/daiya.php)

**例題10:** AさんはA地点を出発してB地点（12km先）へ向かい時速4kmで歩く。BさんはAさんの出発1時間後にB地点を出発してA地点へ向かい時速6kmで歩く。2人が出会うのはAさんが出発してから何時間後か。

<details>
<summary>解答</summary>

Aさんの出発から x 時間後:
Aさんの位置(Aからの距離): 4x
Bさんの位置(Aからの距離): 12 - 6(x-1)（x≧1）

出会うとき: 4x = 12 - 6(x-1)
4x = 12 - 6x + 6
10x = 18, x = 1.8
答え: **1.8時間後（1時間48分後）**

</details>
