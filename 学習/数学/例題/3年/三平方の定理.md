### 基本問題

📖 [三平方の定理 基本問題](https://math.005net.com/reidai/sanheiho.php)

**例題1:** 直角三角形の直角を挟む2辺が 6cm と 8cm のとき、斜辺の長さを求めなさい。

<details>
<summary>解答</summary>

c² = 36 + 64 = 100 → c = **10cm**

</details>

---

📖 [三平方の定理 基本問題2](https://math.005net.com/reidai/sanheiho2.php)

**例題2:** 直角三角形の斜辺が 13cm、一方の辺が 5cm のとき、もう一方の辺の長さを求めなさい。

<details>
<summary>解答</summary>

a² = 13² - 5² = 169 - 25 = 144 → a = **12cm**

</details>

---

📖 [特別な直角三角形　補助線が必要な問題](https://math.005net.com/reidai/tokubetu.php)

**例題3:** 1辺が 6cm の正三角形の高さと面積を求めなさい。

<details>
<summary>解答</summary>

高さ = 6 × √3/2 = **3√3 cm**
面積 = 1/2 × 6 × 3√3 = **9√3 cm²**

</details>

### 平面での利用

📖 [二等辺三角形の面積](https://math.005net.com/reidai/sanheiho_menseki1.php)

**例題4:** AB = AC = 10cm, BC = 12cm の二等辺三角形の面積を求めなさい。

<details>
<summary>解答</summary>

Aから BC に垂線AH を引く。BH = 6cm（中点）
AH² = 10² - 6² = 64 → AH = 8cm
面積 = 1/2 × 12 × 8 = **48cm²**

</details>

---

📖 [台形の面積](https://math.005net.com/reidai/sanheiho_menseki2.php)

**例題5:** 台形ABCDで AD // BC、AD = 5cm, BC = 11cm, AB = CD = 5cm のとき、面積を求めなさい。

<details>
<summary>解答</summary>

A, D から BC に垂線を引く。
BH₁ = (BC - AD)/2 = (11-5)/2 = 3cm
高さ² = 5² - 3² = 16 → 高さ = 4cm
面積 = 1/2 × (5+11) × 4 = **32cm²**

</details>

### 座標

📖 [三平方　2点間の距離](https://math.005net.com/reidai/sanheiho_zahyo0.php)

**例題6:** 2点 A(-1, 3) と B(2, -1) の距離を求めなさい。

<details>
<summary>解答</summary>

AB = √((2-(-1))² + (-1-3)²) = √(9 + 16) = √25 = **5**

</details>

---

📖 [三平方　座標平面の三角形](https://math.005net.com/reidai/sanheiho_zahyo1.php)

**例題7:** 3点 A(0, 4), B(-3, 0), C(5, 0) を頂点とする三角形の面積を求めなさい。

<details>
<summary>解答</summary>

底辺 BC = 5-(-3) = 8（x軸上）
高さ = Aのy座標 = 4
面積 = 1/2 × 8 × 4 = **16**

</details>

### 立体での利用

📖 [直方体の対角線](https://math.005net.com/reidai/chokuhotaiTaikakusen.php)

**例題8:** 縦4cm、横3cm、高さ12cmの直方体の対角線の長さを求めなさい。

<details>
<summary>解答</summary>

対角線 = √(4² + 3² + 12²) = √(16 + 9 + 144) = √169 = **13cm**

</details>

---

📖 [四角錐の体積](https://math.005net.com/reidai/kakusuiTaiseki.php)

**例題9:** 底面が1辺6cmの正方形、すべての辺の長さが6cmの正四角錐の体積を求めなさい。

<details>
<summary>解答</summary>

底面の対角線 = 6√2, 中心から頂点までの距離 = 3√2
高さ² = 6² - (3√2)² = 36 - 18 = 18 → 高さ = 3√2 cm
体積 = 1/3 × 36 × 3√2 = **36√2 cm³**

</details>

---

📖 [最短の道のり](https://math.005net.com/reidai/saitanKeiro.php)

**例題10:** 底面の半径3cm、高さ4cmの円柱の側面上で、底面の円周上の点Aから真上にある上面の円周上の点Bまでの最短距離を求めなさい。

<details>
<summary>解答</summary>

展開図で考える: 側面は横 2π×3 = 6π, 縦 4 の長方形
AとBは横方向に半周分(3π)離れている（真上なので半周）

最短距離 = √((3π)² + 4²) = √(9π² + 16) = **√(9π² + 16) cm**

</details>

📖 [三平方　折り返し](https://math.005net.com/reidai/sanheihoOrikaesi.php)

**例題11:** 長方形ABCDで AB = 8cm, BC = 6cm。頂点Cが辺AB上の点Eに重なるように折り返すとき、AE の長さを求めなさい。

<details>
<summary>解答</summary>

BE = x とおくと AE = 8 - x
折り返しなので CE' = BC の対応辺 → CE' = 6（実際は EC に対応）
折り返し後: E は AB上、折り返した辺の長さ = もとの辺の長さ

直角三角形BEFで: BF² + BE² = EF²
BC = 6 = 折り返し後の辺の長さ
x² + 6² ... ではなく

BE = x, ED'(折り返し) = CD = 8 ではない...
正しくは: EC = 6(もとの辺), BE = x
三平方: x² + 6² ≠ ...

折り返しの点をFとして BF = x, EF = 6(折り返し前のBC)
BF² + AB の一部² = EF²: ではなく

AE = 8 - x, 折り返した辺 CE'=6 にて
AE² + AD² = DE'² → (8-x)² + 36 = ... は別パターン

**正しい解法:** BE = x とし折り返し後 CE → E'、BC = 6 = BE'
右三角形 ABE' で AB=8, BE'=6 → AE' = √(64+36) ...ではなく

折り返し: Cが辺AB上のEに重なる → EC を折り目として
BEの長さをxとすると: 折り返しで BC=BE=6（ではない）

改めて: 折り目をFGとし、C→E。EC=0ではなく...
問題の詳細は 📖 [三平方 折り返し](https://math.005net.com/reidai/sanheihoOrikaesi.php) を参照してください。

</details>
