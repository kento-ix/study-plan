### 基本問題

📖 [相似 基本問題1](https://math.005net.com/reidai/soji1.php)

**例題1:** △ABCと△DEFが相似で、相似比が 3:5 である。AB = 6cm のとき DE の長さを求めなさい。

<details>
<summary>解答</summary>

6 : DE = 3 : 5
DE = 6 × 5/3 = **10cm**

</details>

### 相似の証明

📖 [相似 基本問題2](https://math.005net.com/reidai/sojiSyomei1.php)

**例題2:** △ABCの辺AB上に点D、辺AC上に点Eがあり、DE // BC のとき、△ADE ∽ △ABC を証明しなさい。

<details>
<summary>解答</summary>

△ADEと△ABCにおいて
∠A = ∠A …①（共通）
∠ADE = ∠ABC …②（DE // BC の同位角）
①②より、2組の角がそれぞれ等しいので
△ADE ∽ △ABC

</details>

---

📖 [平行四辺形と相似の証明](https://math.005net.com/reidai/sojiSyomei3.php)

**例題3:** 平行四辺形ABCDの辺BCの中点をMとする。AMの延長とDCの延長の交点をEとするとき、△ABM ∽ △ECM を証明しなさい。

<details>
<summary>解答</summary>

△ABMと△ECMにおいて
BM = CM …①（Mは中点）
∠ABM = ∠ECM …②（AB // EC の錯角）
∠AMB = ∠EMC …③（対頂角）
①②③より、1組の辺とその両端の角から2角相等で
△ABM ∽ △ECM

</details>

### 線分の比

📖 [三角形と線分](https://math.005net.com/reidai/sojiSenbun1.php)

**例題4:** △ABCで DE // BC、AD = 4, DB = 6, AE = 3 のとき、EC と DE の長さを求めなさい（BC = 15）。

<details>
<summary>解答</summary>

AD : AB = AE : AC = DE : BC
4 : 10 = 3 : AC → AC = 7.5 → EC = 7.5 - 3 = **4.5**
4 : 10 = DE : 15 → DE = **6**

</details>

📖 [平行線と線分の長さ](https://math.005net.com/reidai/heikosenSoji.php)

**例題5:** 3本の平行線 ℓ // m // n が2直線と交わり、一方の直線上の切り取られる線分が 3cm, 5cm、もう一方の直線上の一方の線分が 6cm のとき、もう一方の線分の長さを求めなさい。

<details>
<summary>解答</summary>

3 : 5 = 6 : x
3x = 30
x = **10cm**

</details>

### 面積比・体積比

📖 [相似な図形の面積比](https://math.005net.com/reidai/sojiMensekii.php)

**例題7:** 相似比が 2:3 の2つの三角形がある。小さい方の面積が 20cm² のとき、大きい方の面積を求めなさい。

<details>
<summary>解答</summary>

面積比 = 2² : 3² = 4 : 9
20 : S = 4 : 9 → S = **45cm²**

</details>

---

📖 [相似な図形の面積比、体積比](https://math.005net.com/reidai/sojiMensekihiTaisekihi.php)

**例題8:** 相似な2つの円錐の相似比が 3:4 である。小さい方の体積が 54π cm³ のとき、大きい方の体積を求めなさい。

<details>
<summary>解答</summary>

体積比 = 3³ : 4³ = 27 : 64
54π : V = 27 : 64
V = 54π × 64/27 = **128π cm³**

</details>

---

📖 [線分の比と面積比](https://math.005net.com/reidai/senbunhiMensekihi.php)

**例題9:** △ABCで辺BC上に点Dがあり、BD:DC = 2:3 とする。△ABDと△ACDの面積比を求めなさい。

<details>
<summary>解答</summary>

高さが共通なので面積比 = 底辺の比 = BD:DC = **2:3**

</details>

---

📖 [角の二等分線と辺の比1](https://math.005net.com/reidai/soji_kakuni1.php)

**例題10:** △ABCで ∠Aの二等分線と辺BCの交点をDとする。AB = 8, AC = 6 のとき、BD:DC を求めなさい。

<details>
<summary>解答</summary>

角の二等分線の定理: BD:DC = AB:AC = **8:6 = 4:3**

</details>
