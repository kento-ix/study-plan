### 解き方

📖 [2次方程式の解き方(因数分解利用)](https://math.005net.com/reidai/2jihoteisiki_1.php)

**例題1:** 次の方程式を解きなさい。
(1) x² - 5x + 6 = 0
(2) x² - 9 = 0
(3) x² + 4x = 0

<details>
<summary>解答</summary>

(1) (x-2)(x-3) = 0 → **x = 2, 3**
(2) (x+3)(x-3) = 0 → **x = ±3**
(3) x(x+4) = 0 → **x = 0, -4**

</details>

📖 [2次方程式の解き方(解の公式利用)](https://math.005net.com/reidai/2jihoteisiki_2.php)

**例題3:** 次の方程式を解きなさい。
3x² + 5x - 2 = 0

<details>
<summary>解答</summary>

x = (-5 ± √(25+24)) / 6 = (-5 ± 7) / 6
x = 2/6 = 1/3 または x = -12/6 = -2
**x = 1/3, -2**

</details>

**例題4:** 次の方程式を解きなさい。
(x + 1)(x - 3) = 5

<details>
<summary>解答</summary>

(x - 4)(x + 2) = 0
**x = 4, -2**

📖 [置き換えを使って解く2次方程式](https://math.005net.com/reidai/2jihoOkikae.php)

**例題5:** 次の方程式を解きなさい。
(x² + x)² - 8(x² + x) + 12 = 0

<details>
<summary>解答</summary>

A = x² + x とおく: A² - 8A + 12 = 0
(A - 2)(A - 6) = 0 → A = 2 または A = 6

x² + x = 2 → x² + x - 2 = 0 → (x+2)(x-1) = 0 → x = -2, 1
x² + x = 6 → x² + x - 6 = 0 → (x+3)(x-2) = 0 → x = -3, 2
**x = -3, -2, 1, 2**

</details>

### 解から方程式を求める

📖 [解から2次方程式を求める1](https://math.005net.com/reidai/2jiho_kai1.php)

**例題6:** x = 3 が 2x² + ax - 15 = 0 の解のとき、a の値ともう1つの解を求めなさい。

<details>
<summary>解答</summary>

x = 3 を代入: 18 + 3a - 15 = 0 → 3a = -3 → a = -1
2x² - x - 15 = 0 → (2x + 5)(x - 3) = 0
もう1つの解: **x = -5/2**、**a = -1**

</details>

### 文章題

📖 [2次方程式文章題 数の問題](https://math.005net.com/yoten/2jihoBunsyo.php?reidai=rei1)

**例題7:** ある正の整数とその整数より3大きい数の積が108になる。ある正の整数を求めなさい。

<details>
<summary>解答</summary>

x(x + 3) = 108
x² + 3x - 108 = 0
(x + 12)(x - 9) = 0
x = -12, 9　正の整数なので **x = 9**

</details>

📖 [2次方程式文章題 図形の問題](https://math.005net.com/yoten/2jihoBunsyo.php?reidai=rei2)

**例題8:** 長方形の土地の縦を2m長くし、横を3m短くすると面積が6m²減る。もとの長方形の縦が x m、横が(x+5)mのとき、x の値を求めなさい。

<details>
<summary>解答</summary>

もとの面積: x(x+5)
変更後: (x+2)(x+5-3) = (x+2)(x+2)
x(x+5) - (x+2)² = 6
x² + 5x - (x² + 4x + 4) = 6
x - 4 = 6 → **x = 10**

</details>

📖 [2次方程式文章題 道幅に関する問題](https://math.005net.com/reidai/2jihoBunsyo_zu2.php)

**例題9:** 縦20m、横30mの長方形の土地に、同じ幅の道を縦と横に1本ずつ作る。残りの面積が504m²のとき、道の幅を求めなさい。

<details>
<summary>解答</summary>

道幅を x m とすると:
残りの面積 = (20-x)(30-x) = 504
600 - 50x + x² = 504
x² - 50x + 96 = 0
(x - 2)(x - 48) = 0
x = 48 は不適（20mより大きい）
**x = 2m**

</details>

📖 [2次方程式文章題 動点](https://math.005net.com/yoten/2jihoBunsyo.php?reidai=rei3)

**例題10:** 長方形ABCDで AB = 12cm, BC = 8cm。点PがAを出発してA→B→Cの方向に毎秒2cmで動く。出発してx秒後の△APDの面積が24cm²となるのは何秒後か。

<details>
<summary>解答</summary>

【0 ≦ x ≦ 6】PはAB上、AP = 2x
面積 = 1/2 × 8 × 2x = 8x
8x = 24 → x = 3（範囲内 ✓）

【6 < x ≦ 10】PはBC上、BP = 2x - 12
面積 = 1/2 × 8 × 12 - ... (複雑になるため別途計算)

答え: **3秒後**（AB上のとき）

</details>
