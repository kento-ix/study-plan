### 変化の割合

📖 [変化の割合(基本例題)](https://math.005net.com/reidai/2jikan_henka.php)

**例題1:** y = 2x² で、x が 1 から 3 まで変化するときの変化の割合を求めなさい。

<details>
<summary>解答</summary>

公式 a(p+q) = 2(1+3) = **8**

検算: y(1) = 2, y(3) = 18 → (18-2)/(3-1) = 8 ✓

</details>

---

📖 [変化の割合(文字を求める問題)](https://math.005net.com/reidai/2jikan_henka2.php)

**例題2:** y = ax² で、x が -2 から 4 まで変化するときの変化の割合が6のとき、a の値を求めなさい。

<details>
<summary>解答</summary>

a(-2 + 4) = 6
2a = 6
**a = 3**

</details>

### 変域

📖 [変域1(基礎)](https://math.005net.com/reidai/2jikan_heniki1.php)

**例題3:** y = -x² で、-3 ≦ x ≦ 1 のときの y の変域を求めなさい。

<details>
<summary>解答</summary>

x = 0: y = 0（最大値、a < 0 で変域が0をまたぐ）
x = -3: y = -9（最小値）
**-9 ≦ y ≦ 0**

</details>

📖 [変域3(変域から放物線の式を出す)](https://math.005net.com/reidai/2jikan_heniki3.php)

**例題4:** y = ax² で、1 ≦ x ≦ 3 のとき 2 ≦ y ≦ 18 となる。a の値を求めなさい。

<details>
<summary>解答</summary>

a > 0（yが正の範囲）
x = 1: y = a = 2（最小値）→ **a = 2**
確認: x = 3: y = 18 ✓

</details>

### 放物線と直線

📖 [放物線と直線1　交点を出す](https://math.005net.com/reidai/2jikan_koten1.php)

**例題5:** y = x² と y = 2x + 3 の交点の座標を求めなさい。

<details>
<summary>解答</summary>

x² = 2x + 3
x² - 2x - 3 = 0
(x - 3)(x + 1) = 0
x = 3: y = 9, x = -1: y = 1
交点: **(-1, 1) と (3, 9)**

</details>

---

📖 [放物線と直線3　点から式を出す](https://math.005net.com/reidai/2jikan_koten3.php)

**例題6:** y = ax² のグラフと直線 y = x + 6 が点A(-2, 4) で交わるとき、a の値とも う1つの交点Bの座標を求めなさい。

<details>
<summary>解答</summary>

A(-2, 4): 4 = a(-2)² → 4 = 4a → **a = 1**
y = x² と y = x + 6: x² = x + 6
x² - x - 6 = 0 → (x-3)(x+2) = 0
x = 3: y = 9
もう1つの交点: **B(3, 9)**

</details>

### 放物線と図形

📖 [放物線と図形　三角形の面積](https://math.005net.com/reidai/2jikan_menseki.php)

**例題7:** y = x² のグラフ上に点A(-2, 4)、点B(3, 9) がある。直線ABとy軸の交点をCとするとき、△AOBの面積を求めなさい（Oは原点）。

<details>
<summary>解答</summary>

直線AB: 傾き = (9-4)/(3-(-2)) = 1, 切片 → y = x + 6, C(0, 6)
△AOB = △AOC + △BOC
△AOC: 底辺OC = 6, 高さ = |-2| = 2 → 面積 = 1/2 × 6 × 2 = 6
△BOC: 底辺OC = 6, 高さ = |3| = 3 → 面積 = 1/2 × 6 × 3 = 9
△AOB = 6 + 9 = **15**

</details>

📖 [放物線と図形　正方形](https://math.005net.com/reidai/2jikan_seihokei.php)

**例題8:** y = x² のグラフ上に2点A, Bがあり、ABをx軸に平行な辺とする正方形ABCDがx軸の下にある。この正方形の1辺の長さを求めなさい。

<details>
<summary>解答</summary>

A(-a, a²), B(a, a²) とすると AB = 2a
正方形なので1辺 = 2a、Aのy座標 = a² = 2a
a² - 2a = 0 → a(a-2) = 0 → a = 2（a > 0）
1辺 = **4**

</details>

📖 [動点](https://math.005net.com/reidai/doten3nen.php)

**例題9:** 直角三角形ABCで ∠B = 90°, AB = 6cm, BC = 8cm。点PがBを出発してB→Cの方向に毎秒1cmで進む。x秒後の △ABP の面積を y cm² とするとき、y を x の式で表しなさい。

<details>
<summary>解答</summary>

BP = x, 底辺BP = x, 高さAB = 6
y = 1/2 × x × 6 = **3x**（0 ≦ x ≦ 8）

</details>
