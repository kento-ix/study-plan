### 式の展開

📖 [式の展開](https://math.005net.com/reidai/tenkai.php)

**例題1:** 次の式を展開しなさい。
(3x - 2)(x + 4)

<details>
<summary>解答</summary>

= 3x² + 12x - 2x - 8 = **3x² + 10x - 8**

</details>

---

📖 [乗法公式(x+a)(x+b)の展開](https://math.005net.com/reidai/johokosiki1.php)

**例題2:** 次の式を展開しなさい。
(x + 3)(x - 7)

<details>
<summary>解答</summary>

= x² + (3-7)x + 3×(-7) = **x² - 4x - 21**

</details>

📖 [乗法公式 和と差の積の展開](https://math.005net.com/reidai/johokosiki3.php)

📖 [式の展開 おきかえ](https://math.005net.com/reidai/tenkaiOkikae.php)

**例題5:** 次の式を展開しなさい。
(x + y + 3)(x + y - 3)

<details>
<summary>解答</summary>

A = x + y とおくと: (A + 3)(A - 3) = A² - 9
= (x + y)² - 9 = **x² + 2xy + y² - 9**

</details>

### 因数分解

📖 [因数分解1 共通因数をくくりだす](https://math.005net.com/reidai/inbun01.php)

**例題6:** 次の式を因数分解しなさい。
6x²y - 9xy²

<details>
<summary>解答</summary>

= **3xy(2x - 3y)**

</details>

---

📖 [因数分解2 (x+a)(x+b)](https://math.005net.com/reidai/inbun02.php)

**例題7:** 次の式を因数分解しなさい。
x² + 7x + 12

<details>
<summary>解答</summary>

積が12、和が7 → 3と4
= **(x + 3)(x + 4)**

</details>

📖 [因数分解4 (x+a)(x-a)](https://math.005net.com/reidai/inbun04.php)

**例題9:** 次の式を因数分解しなさい。
9x² - 16

<details>
<summary>解答</summary>

= (3x)² - 4² = **(3x + 4)(3x - 4)**

</details>

---

📖 [因数分解 おきかえ](https://math.005net.com/reidai/inbun05.php)

**例題10:** 次の式を因数分解しなさい。
(x + 2)² - 5(x + 2) + 6

<details>
<summary>解答</summary>

A = x + 2 とおく: A² - 5A + 6 = (A - 2)(A - 3)
= (x + 2 - 2)(x + 2 - 3) = **x(x - 1)**

</details>

---

📖 [共通因数をくくりだした後さらに因数分解](https://math.005net.com/reidai/inbun06.php)

**例題11:** 次の式を因数分解しなさい。
3x² - 12

<details>
<summary>解答</summary>

= 3(x² - 4) = **3(x + 2)(x - 2)**

</details>

### 応用

📖 [数の計算のくふう](https://math.005net.com/reidai/keisanRiyo.php)

**例題12:** 因数分解を利用して計算しなさい。
53² - 47²

<details>
<summary>解答</summary>

= (53 + 47)(53 - 47) = 100 × 6 = **600**

</details>

📖 [数の性質の証明](https://math.005net.com/reidai/sikinoSyomei3nen.php)

**例題13:** 連続する2つの偶数の積に1を足すと奇数の2乗になることを証明しなさい。

<details>
<summary>解答</summary>

連続する2つの偶数を 2n, 2n+2 とする。
2n(2n + 2) + 1 = 4n² + 4n + 1 = (2n + 1)²
2n+1 は奇数なので、(2n+1)² は奇数の2乗である。

</details>
