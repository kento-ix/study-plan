# 数学 要点別学習計画

> 参考: [中学数学 要点のまとめ](https://math.005net.com/youten.php)（習和学院）

神奈川県公立高校入試の出題傾向に基づき、中学3年間の数学要点を入試対策として整理した学習計画です。

---

## 学習の進め方

1. 各単元の要点を上記サイトで確認する
2. 理解度チェックで自己評価する（◎○△×）
3. △×の単元を重点的に演習する
4. 過去問で実践力を確認する

## 入試に向けた重点学習スケジュール

### Phase 1: 基礎完成（中3・4月〜7月）
**目標:** 1年〜2年の全単元で理解度「○」以上

- 上の表で理解度が△×の1年・2年単元を優先的に復習
- 特に重要: **計算力**（正負の数〜平方根）、**方程式の文章題**、**合同の証明**
- 毎日の計算ドリル（15分）を習慣にする

### Phase 2: 3年範囲の習得（中3・8月〜10月）
**目標:** 3年範囲の全単元で理解度「○」以上

- 因数分解・平方根・2次方程式を最優先
- 関数 y=ax² と1次関数の融合問題に取り組む
- 相似・円周角・三平方の定理を順番に学習

### Phase 3: 入試演習（中3・11月〜1月）
**目標:** 過去問で安定して70点以上

- 神奈川県過去問を年度別に演習
- 間違えた問題を単元別に分類し、上の表の理解度を更新
- 弱点単元はサイトの要点ページに戻って復習

### Phase 4: 直前仕上げ（中3・2月）
**目標:** 本番で80点以上

- 時間配分の練習（50分で解ききる訓練）
- 問1・問2の39点を確実に取る演習
- 記述問題（証明・確率）の答案の書き方を仕上げる
