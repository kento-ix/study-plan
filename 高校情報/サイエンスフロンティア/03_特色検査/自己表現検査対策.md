# 特色検査（自己表現検査）対策

## 検査概要

- **時間:** 60分間
- **形式:** 記述式
- **内容:** 理科・数学・英語など各教科の知識を活用した教科横断型の問題

## 評価の観点

| 観点 | 内容 |
|------|------|
| 読解力 | 提示された資料を正確に読み取る力 |
| 課題設定力 | 問題の本質を見抜き、課題を明確にする力 |
| 情報活用力 | 複数の資料から必要な情報を選び、整理する力 |
| 課題解決力 | 論理的に考え、解決策を記述する力 |

## 対策方法

### 1. 過去問演習
- サイエンスフロンティア高校の過去の特色検査問題を解く
- 神奈川県の他の特色検査実施校の問題にも取り組む

### 2. 教科横断的な学習
- 理科の実験データをグラフ化し、数学的に分析する練習
- 英語の科学記事を読み、内容を日本語で要約する練習
- 社会問題を科学的視点から考察する練習

### 3. 記述力の強化
- 結論を先に述べ、根拠を示す書き方を練習する
- 図やグラフを用いた説明の練習
- 制限時間内に書き上げる練習
