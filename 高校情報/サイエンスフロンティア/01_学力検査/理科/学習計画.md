# 理科 学習計画（学力検査 重点化2倍）

## 重要度: ★★★★★（最優先）

学力検査・調査書ともに2倍の重点化。数学と並ぶ最重要教科。

---

## 出題構成と配点（令和7年度実績）

> 出典: [神奈川県公立高等学校入学者選抜 学力検査問題・正答表（令和7年度）](https://www.pref.kanagawa.jp/docs/dc4/nyusen/nyusen/gakuryokukensa/r7.html)

| 大問 | 分野 | 内容 | 配点 |
|------|------|------|------|
| 問1 | 物理 | 光の屈折（凸レンズ）・電流と抵抗・静電気 | 9点 |
| 問2 | 化学 | 密度と浮力・電離のモデル・中和とイオン | 9点 |
| 問3 | 生物 | 顕微鏡の使い方・動物の分類・血液循環 | 9点 |
| 問4 | 地学 | 露点と霧の実験・太陽系の惑星・極夜 | 9点 |
| 問5 | 物理 | 斜面上の小球の運動・力の分解・速さの計算 | 16点 |
| 問6 | 化学 | 炭酸水素ナトリウムとクエン酸の反応・質量保存の法則 | 16点 |
| 問7 | 生物 | 微生物によるデンプンの分解・対照実験・分解者 | 16点 |
| 問8 | 地学 | プレートテクトニクス・地震計のしくみ・P波S波と緊急地震速報 | 16点 |

### 分野別配点まとめ

| 分野 | 配点 | 大問 |
|------|------|------|
| 物理 | **25点** | 問1（9点）+ 問5（16点） |
| 化学 | **25点** | 問2（9点）+ 問6（16点） |
| 生物 | **25点** | 問3（9点）+ 問7（16点） |
| 地学 | **25点** | 問4（9点）+ 問8（16点） |

**4分野が完全に均等配点（各25点）。苦手分野を作らないことが重要。**

### 出題の特徴
- 各分野で小問集合（9点）+ 実験・観察の大問（16点）の2問構成
- 小問集合は基礎知識、大問は実験の読み取り・考察が中心
- 計算問題は少なめだが、グラフの読み取りが頻出

---

## 学習ステップ

### Step 1: 基礎知識の定着（小問集合36点を確実に）
- [ ] 各分野の重要用語・法則を整理する
- [ ] 教科書の実験内容と結果を確認する
- [ ] 化学式・化学反応式を暗記する
- [ ] 公式（オームの法則、密度、速さなど）を覚える

### Step 2: 分野別の強化
- [ ] **物理:** 力の分解（斜面の問題）、電流の計算、光の屈折
- [ ] **化学:** 質量保存の法則を使った計算、イオンと中和のグラフ
- [ ] **生物:** 対照実験の考え方、分類のポイント、血液循環のしくみ
- [ ] **地学:** P波S波の速さの計算、プレートの動き、天気の変化

### Step 3: 実験・観察問題の対策（大問64点の対策）
- [ ] 実験の目的・手順・結果・考察を説明できるようにする
- [ ] グラフや表の読み取り練習
- [ ] 「対照実験」の設計理由を説明できるようにする
- [ ] 実験結果から結論を導く記述問題の練習

### Step 4: 過去問演習
- [ ] 神奈川県公立入試の過去問を5年分以上解く
- [ ] 時間配分の目安: 小問集合4問(15分) → 大問4問(各8分=32分) → 見直し(3分)
- [ ] 苦手分野の集中復習
