# 数学 学習計画（学力検査 重点化2倍）

## 重要度: ★★★★★（最優先）

学力検査・調査書ともに2倍の重点化。最も配点に影響する教科。

---

## 出題構成と配点（令和7年度実績）

> 出典: [神奈川県公立高等学校入学者選抜 学力検査問題・正答表（令和7年度）](https://www.pref.kanagawa.jp/docs/dc4/nyusen/nyusen/gakuryokukensa/r7.html)

| 大問 | 内容 | 配点 | 形式 |
|------|------|------|------|
| 問1 | 基礎計算（正負の数・分数・文字式・平方根） | 15点（5問×3点） | 選択 |
| 問2 | 基本問題（因数分解・二次方程式・変化の割合・文章題・平方根の応用・回転体） | 24点（6問×4点） | 選択 |
| 問3 | 図形の証明・箱ひげ図・方程式の応用・角度 | 25点 | 選択+記述 |
| 問4 | 関数（二次関数・反比例・一次関数の融合問題） | 15点 | 選択+記述 |
| 問5 | 確率 | 11点（5点+6点） | 記述 |
| 問6 | 空間図形（三角柱・表面積・垂線の長さ） | 10点（4点+6点） | 選択+記述 |

### 分野別配点まとめ

| 分野 | 配点 | 備考 |
|------|------|------|
| 数と式・計算 | 約25点 | 問1全体 + 問2の一部（確実な得点源） |
| 図形 | 約35点 | 問3（証明含む）+ 問6（空間図形）+ 問2の一部 |
| 関数 | 約15点 | 問4（複数関数の融合、後半は難問） |
| データの活用・確率 | 約16点 | 問3イ（箱ひげ図）+ 問5（確率） |
| 方程式の文章題 | 約9点 | 問2エ + 問3ウ |

## 学習ステップ

### Step 1: 基礎固め（問1・問2で39点を確実に取る）
- [ ] 計算力の強化（正負の数〜平方根まで毎日10分）
- [ ] 因数分解・二次方程式の解法を確実にする
- [ ] 変化の割合・文章題・平方根の応用問題に慣れる
- [ ] 回転体の体積の求め方を練習する

### Step 2: 図形の強化（最大配点35点の分野）
- [ ] 円周角の定理・相似の証明を自力で書けるようにする
- [ ] 三平方の定理を使った計算問題
- [ ] 空間図形（三角柱・切断面・垂線）の問題演習
- [ ] 角度を求める問題（正方形+直角三角形の組み合わせ等）

### Step 3: 関数・確率の演習
- [ ] 一次関数・二次関数・反比例の融合問題
- [ ] 座標上の図形の面積比の問題
- [ ] 確率の応用問題（さいころ2つ・条件付き確率）
- [ ] 箱ひげ図の読み取りと四分位数の理解

### Step 4: 過去問演習
- [ ] 神奈川県公立入試の過去問を5年分以上解く
- [ ] 時間配分の目安: 問1(5分) → 問2(10分) → 問3(12分) → 問4(8分) → 問5(8分) → 問6(7分)
- [ ] 間違えた問題の解き直しノートを作る

---

## 得点戦略

- **確実に取る:** 問1(15点) + 問2(24点) = 39点
- **高得点を狙う:** 問3前半 + 問5ア = 約15点
- **差がつく問題:** 問4ウ(6点)・問6イ(6点) → 記述式で部分点なし、余裕があれば挑戦

- 休日: 90〜120分
