# 国語 学習計画

## 重要度: ★★★☆☆（標準）

重点化なし。確実に得点できる基礎力を身につける。

## 出題構成と配点（令和7年度実績）

> 出典: [神奈川県公立高等学校入学者選抜 学力検査問題・正答表（令和7年度）](https://www.pref.kanagawa.jp/docs/dc4/nyusen/nyusen/gakuryokukensa/r7.html)

| 大問 | 内容 | 配点 | 形式 |
|------|------|------|------|
| 問一 | 漢字の読み(4問)・漢字の書き(4問)・文法 | 20点 | 記述+選択 |
| 問二 | 文学的文章の読解（小説・随筆） | 24点（6問×4点） | 選択 |
| 問三 | 説明的文章の読解（論説文） | 30点（9問） | 選択 |
| 問四 | 古典（古文）の読解 | 20点（5問×4点） | 選択 |
| 問五 | 記述問題（条件付き作文） | 6点 | 記述 |

### 分野別配点まとめ

| 分野 | 配点 | 備考 |
|------|------|------|
| 漢字・文法 | **20点** | 問一（確実な得点源） |
| 文学的文章 | **24点** | 問二（心情の読み取りが中心） |
| 説明的文章 | **30点** | 問三（最大配点、論理的読解力） |
| 古典 | **20点** | 問四（古文の基礎が中心） |
| 記述 | **6点** | 問五（指定語句あり、30〜40字程度） |

### 出題の特徴
- 説明的文章（問三）が30点で最大の配点
- 記述問題は問五のみ（6点）。指定語句を含めて書く条件付き
- 漢字は読み4問・書き4問の計8問（各2点=16点）で得点しやすい

## 学習ステップ

### Step 1: 基礎固め（問一で20点を確実に）
- [ ] 漢字の読み書き（中学配当漢字を中心に）を毎日練習
- [ ] 文法の基本事項（品詞の分類、助詞・助動詞）を確認
- [ ] 古文の基本単語・歴史的仮名遣いを覚える

### Step 2: 読解力の強化（問二+問三で54点）
- [ ] 説明的文章：筆者の主張と根拠を正確に読み取る練習
- [ ] 文学的文章：登場人物の心情変化を追う練習
- [ ] 接続語・指示語に注目して文章構造を把握する
- [ ] 選択肢問題の消去法を身につける

### Step 3: 古典・記述問題の対策
- [ ] 古文の基本的な読み方（主語の把握・現代語訳）
- [ ] 「なぜ〜か」「どういうことか」の記述パターンを練習
- [ ] 指定語句を含めた条件付き記述の練習
- [ ] 字数制限のある記述問題に慣れる

### Step 4: 過去問演習
- [ ] 神奈川県公立入試の過去問を5年分以上解く
- [ ] 時間配分の目安: 問一(5分) → 問二(10分) → 問三(15分) → 問四(10分) → 問五(8分) → 見直し(2分)
- [ ] 古典の問題を重点的に復習
