# 中学生 学習計画リポジトリ

横浜サイエンスフロンティア高等学校（令和8年度入学者選抜）の受験に向けた学習計画・進捗管理リポジトリです。

## ディレクトリ構成

```
study-plan/
├── README.md              ← このファイル
├── 学習/                   ← 学力検査対策（教科別の学習計画・過去問・教材）
│   └── 01_学力検査/
│       ├── 数学/           ← 重点化2倍（最優先）
│       ├── 理科/           ← 重点化2倍（最優先）
│       ├── 英語/
│       ├── 国語/
│       └── 社会/
├── 高校情報/               ← 志望校の選考基準・特色検査・スケジュール等の情報
│   └── サイエンスフロンティア/
├── 進捗/                   ← 学習の進捗記録
└── .gitignore
```

## 志望校

**横浜サイエンスフロンティア高等学校**

- 学力検査の比率が最も高い（第1次:7、第2次:8）
- 数学・理科が2倍に重点化
- 特色検査あり（自己表現検査 60分）

## 対策の優先順位

1. **最優先:** 数学・理科（学力検査2倍 + 調査書2倍）
2. **重要:** 英語（調査書2倍）
3. **標準:** 国語・社会
4. **並行対策:** 特色検査（教科横断型・記述式）

## 使い方

1. `学習/` の各教科フォルダで学習計画を確認し、日々の勉強を進める
2. `高校情報/` で志望校の選考基準や出題傾向を確認する
3. `進捗/` に学習の進捗状況を記録する
